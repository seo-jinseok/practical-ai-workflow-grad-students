## 📖 프로젝트 소개

대학원생을 위한 **실용적 AI 워크플로우**는 생성형 AI를 연구 파트너로 활용하여 연구 생산성을 10배 향상시키는 체계적인 방법을 제공합니다.

### 🎯 주요 특징

- **📚 3부작 구조**: 기초 → 고급 → 통합 워크플로우
- **🔬 전공 무관**: 인문·사회·자연·공학 모두 적용 가능(코딩 지식 불필요)
- **📊 실습 중심**: 4개의 워크북 + 8개 데모 시나리오
- **🚀 주 사용 도구(무료 도구 중심)**: Visual Studio Code, GitHub Copilot(다른 생성형 AI 확장도 가능), Claude Desktop, MCP, SpecKit, Elicit 등

### 📈 기대 효과

- **연구 효율성**: 체계적 워크플로우로 연구를 위한 업무 시간 절약
- **AI 활용 능력**: AI를 연구 파트너로 전환
- **문서화 역량**: Markdown 기반 연구 문서 체계 구축
