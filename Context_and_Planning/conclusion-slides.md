# 마무리 (Conclusion) - 상세 슬라이드 콘텐츠

이 문서는 마무리 5분 분량의 슬라이드를 상세히 정의합니다.

## 슬라이드 개요
- **총 슬라이드 수**: 5장 (최적화 완료)
- **소요 시간**: 5분 (Q&A와 마무리 통합으로 조정)
- **학습 목표**: 핵심 내용 정리, 즉시 실천 가능한 액션 도출, 지속적 학습 리소스 확보
- **핵심 전략**: 간결한 요약, 구체적 실천 가이드, 동기부여 메시지로 마무리

## 슬라이드 1: 전체 요약 - 3가지 핵심
**타입**: 요약 슬라이드
**소요 시간**: 1분 (시간 절약)

### 콘텐츠
**제목**: "오늘 배운 3가지 핵심 💎"

**1. Context Engineering** 💬 (20pt)
**핵심 원칙**:
- ✓ 명확하고 구체적으로 (5W1H)
- ✓ Markdown으로 구조화
- ✓ 충분한 컨텍스트 제공
- ✓ 예시 기반 학습 (Few-shot)

**기억할 문장**: "좋은 질문이 좋은 답을 만든다"

**2. Planning & Task Management** 📋 (20pt)
**핵심 원칙**:
- ✓ Spec 문서로 명확한 계획
- ✓ 관리 가능한 단위로 분해
- ✓ AI로 작업 자동 제안
- ✓ 3단계 검증 (AI + 자가 + 전문가)

**기억할 문장**: "명확한 계획이 성공적인 실행을 만든다"

**3. Integrated Workflow** 🔄 (20pt)
**핵심 원칙**:
- ✓ Context 설정 → Planning → 실행 → 검증
- ✓ 각 단계에서 AI와 협업
- ✓ 지속적인 대화와 개선
- ✓ 도구가 아닌 원칙 중심

**기억할 문장**: "AI는 도구가 아니라 협업 파트너"

**전체 핵심 메시지** (28pt, 강조색, 중앙):
```
명확한 Context
    +
체계적 Planning
    +
지속적 Verification
    =
성공적인 연구 프로젝트
```

### 시각 요소
- 3개 박스를 세로로 배치
- 각 박스에 대표 아이콘과 색상
- 핵심 메시지는 수식 스타일로 하단 강조
- 체크마크는 녹색

### 발표자 노트
- "2시간 동안 많은 내용을 배웠습니다"
- "하지만 핵심은 이 3가지입니다"
- "각각의 기억할 문장을 꼭 기억하세요"
- "이 3가지만 실천해도 연구 방식이 완전히 달라집니다"
- 천천히, 명확하게 각 항목 강조

## 슬라이드 2: 실천 가이드 - 오늘부터 시작
**타입**: 콘텐츠 슬라이드
**소요 시간**: 1분 (시간 절약)

### 콘텐츠
**제목**: "🚀 오늘부터 시작하는 3단계 실천 가이드"

**Level 1: 오늘 저녁 (30분 투자)** 🌱 (18pt)
```markdown
1. 현재 프로젝트를 Markdown으로 정리
   - 연구 질문, 목표, 방법을 구조화
   - 파일로 저장 (research-project.md)

2. AI에게 첫 질문
   - "이 연구 프로젝트를 검토하고 개선점을 제안해줘"
   - 응답을 같은 파일에 추가

3. 배운 내용 복습
   - 강의 자료 다시 보기
   - 핵심 3가지 메모
```

**Level 2: 이번 주 (2시간 투자)** 🌿 (18pt)
```markdown
1. Spec 문서 작성
   - 프로젝트 목표, 범위, 일정, 산출물
   - AI와 대화하며 정교화

2. 작업 분해 요청
   - "이 프로젝트를 완료 가능한 작업들로 분해해줘"
   - 우선순위 설정

3. 첫 작업 완료
   - 가장 작은 작업 하나를 AI와 함께 완료
   - 검증 프로세스 적용
```

**Level 3: 이번 달 (지속적)** 🌳 (18pt)
```markdown
1. 주간 루틴 확립
   - 매주 금요일 진행 상황 점검
   - AI에게 다음 주 계획 제안 받기

2. 검증 습관화
   - 모든 AI 출력물 검토
   - 체크리스트 활용

3. 워크플로우 개선
   - 자신에게 맞게 조정
   - 효과적인 부분 강화
```

**약속** (24pt, 강조색):
"1개월 후, 여러분의 연구 방식이 완전히 달라져 있을 것입니다!"

### 시각 요소
- 3단계를 계단식으로 배치
- 각 레벨에 식물 성장 아이콘 (씨앗→새싹→나무)
- 코드 블록 스타일로 구체적 액션
- 약속은 하단 중앙 강조

### 발표자 노트
- "완벽하게 시작하려 하지 마세요"
- "오늘 저녁 30분만 투자하면 시작할 수 있습니다"
- "작은 성공을 경험하면 자연스럽게 확장됩니다"
- "1개월 후 여러분은 AI를 자연스럽게 활용하고 있을 겁니다"
- "저도 처음엔 어색했지만 지금은 필수 도구가 되었습니다"

## 추가 참고사항

### 시간 관리
- 각 슬라이드의 예상 시간을 엄수
- 총 5분 중 30초는 버퍼로 확보
- Q&A는 시간에 따라 유연하게 조정
- 시간 부족 시 슬라이드 4 축약 가능

### 마무리 전략
- 긍정적이고 동기부여되는 분위기
- 구체적이고 실천 가능한 액션 제시
- 지속적 연결 고리 제공 (이메일, GitHub)
- 감사와 격려로 마무리

### 청중 참여
- 슬라이드 5에서 질의응답
- 시간 여유 있으면 성공 다짐 받기
- "1주일 챌린지 하실 분?" (손들기)

### 에너지 관리
- 요약(슬라이드 1-2)은 명확하고 간결하게
- 실천 가이드(슬라이드 2-4)는 구체적이고 동기부여
- 마무리(슬라이드 6)는 따뜻하고 격려하는 톤
- 전체적으로 긍정적이고 희망적인 분위기 유지

### 부산 지역 연결
- 슬라이드 4에서 부산 지역 스터디 그룹 제안
- 지역 대학 간 네트워킹 기회 언급
- 부산 지역 AI 커뮤니티 소개 (있는 경우)

### 후속 조치
- 강의 후 1주일 내 자료 공유 (이메일)
- 1개월 후 팔로업 이메일 (진행 상황 확인)
- 성공 사례 수집 및 공유
- 다음 강의 또는 워크샵 안내
