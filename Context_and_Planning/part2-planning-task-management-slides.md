## 슬라이드 5: Spec-driven Planning 개념
**타입**: 콘텐츠 슬라이드
**소요 시간**: 2분

### 콘텐츠
**제목**: "Spec-driven Planning이란?"

**정의** (18pt):
프로젝트의 **목표, 범위, 요구사항**을 명확히 정의한 문서(Spec)를
기반으로 계획을 수립하고 실행하는 방법론

**핵심 요소** (16pt):
- **Spec (명세서)**: 프로젝트의 청사진
- **문서화**: 모든 것을 기록
- **추적 가능성**: 변경 이력 관리
- **AI 친화적**: 명확한 입력 → 명확한 출력

**비유** (16pt):
건축에서 설계도 없이 집을 짓지 않듯,
연구도 명세 없이 시작하지 마세요

### 시각 요소
- 설계도 → 건물 비유 일러스트
- 핵심 요소는 박스로 구분

### 발표자 노트
- "Spec은 프로젝트의 나침반입니다"
- "특히 AI와 협업할 때 명확한 Spec이 필수적입니다"

## 슬라이드 6: 전통적 방식 vs Spec-driven
**타입**: 비교 슬라이드
**소요 시간**: 2분

### 콘텐츠
**제목**: "전통적 방식 vs Spec-driven Planning"

**전통적 방식** (좌측, 배경 #FEE2E2):
1. 머릿속으로 대충 계획 💭
2. 바로 작업 시작 🏃
3. 중간에 방향 상실 😵
4. 뒤늦게 요구사항 변경 🔄
5. 일정 지연 및 품질 저하 📉

**결과**: 혼란, 재작업, 스트레스

**Spec-driven 방식** (우측, 배경 #D1FAE5):
1. Spec 문서 작성 📝
2. 작업 분해 및 계획 🗂️
3. 명확한 방향성 유지 🎯
4. 진행 상황 추적 📊
5. 체계적 완료 ✅

**결과**: 명확성, 효율성, 품질

### 시각 요소
- 좌우 비교 레이아웃
- 전통적 방식은 빨간 배경, Spec-driven은 녹색 배경
- 각 단계에 이모지 아이콘

### 발표자 노트
- "여러분은 어느 쪽에 가까우신가요?"
- "Spec-driven은 처음에 시간이 걸려도 전체적으로 훨씬 효율적입니다"
- "특히 AI와 협업할 때 그 차이가 극명합니다"

## 슬라이드 7: Spec-driven의 장점 (선택사항)
**타입**: 콘텐츠 슬라이드
**소요 시간**: 2분 (시간 부족 시 축약 가능)

### 콘텐츠
**제목**: "Spec-driven Planning의 5가지 장점"

**장점** (18pt):
1. **명확성** 🎯
   - 모든 이해관계자가 같은 목표 공유
   - 모호함 제거

2. **추적 가능성** 📊
   - 진행 상황을 정량적으로 측정
   - 문제 조기 발견

3. **협업 용이성** 🤝
   - 지도교수, 동료와 쉽게 공유
   - 피드백 반영 용이

4. **AI 활용 최적화** 🤖
   - 명확한 Spec → 정확한 AI 지원
   - 자동화 가능성 증가

5. **재사용성** ♻️
   - 다음 프로젝트의 템플릿
   - 학습 자산 축적

### 시각 요소
- 5개 박스를 카드 형식으로 배치
- 각 장점에 아이콘

### 발표자 노트
- 각 장점을 간단히 설명 (각 20초)
- "특히 4번, AI 활용 최적화가 오늘의 핵심입니다"
- "명확한 Spec이 있어야 AI가 제대로 도와줄 수 있습니다"

```markdown
# 데이터 분석 Spec

## 목표
설문 데이터에서 학습 전략과 성취도의 관계 분석

## 데이터
- 설문 응답: n=200
- 변수: 10개 (독립 3, 종속 2, 통제 5)

## 분석 방법
- 기술 통계
- 상관 분석
- 다중 회귀 분석

## 산출물
- [ ] 데이터 정제 스크립트
- [ ] 분석 스크립트
- [ ] 결과 표 3개
- [ ] 결과 그래프 5개
- [ ] Results 섹션 초안
```

**AI 활용** (16pt):
1. **코드 작성**: GitHub Copilot으로 분석 스크립트
2. **결과 해석**: AI에게 통계 결과 해석 요청
3. **시각화**: 적절한 그래프 유형 제안
4. **작성**: Results 섹션 초안 생성

**소요 시간** (16pt):
- 전통적 방법: 2-3주
- AI + Spec-driven: 1주
