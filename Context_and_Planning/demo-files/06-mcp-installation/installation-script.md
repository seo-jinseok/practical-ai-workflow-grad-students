# MCP 설치 시연 스크립트 (5분)
## Claude Desktop + task-master-mcp

**목적**: 초보자도 따라할 수 있는 MCP 설치 과정 시연  
**대상**: 터미널 사용 경험이 제한적인 학생들  
**전제**: Claude Desktop이 이미 설치되어 있음 (사전 준비)  

## Step 1: Claude Desktop 확인 (1분)

### 강사 액션
1. Claude Desktop 실행
2. 로그인 상태 확인
3. 새 대화 시작

### 강사 멘트
"MCP를 사용하려면 먼저 Claude Desktop이 필요합니다.
이미 설치되어 있다고 가정하겠습니다.
설치 방법은 자료에 상세히 나와 있습니다.

[화면 공유]
보시다시피 Claude Desktop이 실행되었습니다.
이제 MCP 서버를 연결하겠습니다."

### 화면
- Claude Desktop 메인 화면
- 새 대화 창

### 트러블슈팅
- **Claude 미설치 시** → "설치는 자료 참고, 오늘은 설정 과정만" 설명

## Step 2: 설정 파일 편집 (2분)

### 강사 액션
1. VS Code 실행
2. 설정 파일 경로 설명
3. `claude_desktop_config.json` 파일 열기 (미리 준비)
4. task-master-mcp 설정 추가

### 강사 멘트
"MCP 서버를 추가하려면 설정 파일을 편집해야 합니다.

[화면 전환: VS Code]

파일 위치는:
- macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
- Windows: %APPDATA%\Claude\claude_desktop_config.json

[파일 열기]

JSON 파일입니다. 코딩을 모르셔도 괜찮습니다.
복사-붙여넣기만 하면 됩니다.

[설정 추가]

이 부분을 추가합니다:
```json
{
  "mcpServers": {
    "task-master-mcp": {
      "command": "npx",
      "args": ["-y", "task-master-ai"]
    }
  }
}
```

중괄호, 쉼표, 따옴표를 정확히 입력해야 합니다.
자료에 복사할 수 있는 코드가 있으니 걱정 마세요.

[파일 저장: Cmd/Ctrl + S]

저장했습니다."

### 화면
- VS Code에서 JSON 파일 편집
- 문법 강조 표시
- 저장 확인

### 트러블슈팅
- **JSON 문법 오류** → VS Code가 빨간 밑줄 표시, 수정 필요
- **파일 찾기 어려움** → 미리 열어둔 파일 사용

## Step 3: Claude 재시작 및 확인 (1분)

### 강사 액션
1. Claude Desktop 완전 종료 (Cmd+Q 또는 작업 관리자)
2. 재시작
3. 새 대화 시작
4. 테스트 프롬프트 입력

### 강사 멘트
"설정을 적용하려면 Claude를 재시작해야 합니다.

[Claude 종료 및 재시작]

이제 MCP가 작동하는지 확인하겠습니다.

[새 대화 시작]

간단한 질문을 해보겠습니다:
'task-master-ai를 사용할 수 있나요?'

[AI 응답 대기]

보시다시피 AI가 task-master-ai 도구를 인식합니다.
설치 성공입니다!"

### 테스트 프롬프트
```
task-master-mcp를 사용할 수 있나요?
```

### 예상 AI 응답
```
네, task-master-mcp를 사용할 수 있습니다. 
이 도구는 프로젝트 관리와 작업 추적을 도와드립니다.
어떤 프로젝트를 관리하고 싶으신가요?
```

### 트러블슈팅
- **MCP 인식 안 됨** → 설정 파일 재확인, 재시작 다시
- **여전히 실패** → "기술적 문제, 백업 자료로" 전환

## Step 4: 간단한 사용 예시 (1분)

### 강사 액션
1. 간단한 프로젝트 정보 입력
2. AI 응답 확인
3. 마무리

### 강사 멘트
"실제로 사용해보겠습니다.

[프롬프트 입력]

'새 프로젝트: 석사 논문, 6개월, 현재 문헌 조사 중. 다음 작업 3개 제안해줘.'

[AI 응답 확인]

보시다시피 AI가 프로젝트를 분석하고 다음 작업을 제안합니다.
이것이 task-master-mcp의 기본 사용법입니다.

더 자세한 사용법은 Part 2 자료를 참고하세요."

### 마무리 멘트
"MCP 설치가 어렵다면 대안이 있습니다:
- ChatGPT Custom GPTs로 프로젝트 관리 GPT 만들기
- Claude Projects 기능 사용
- Notion AI 활용

중요한 것은 도구가 아니라 체계적인 접근 방식입니다.
Context Engineering + Planning 원칙만 지키면 어떤 도구로도 가능합니다."
