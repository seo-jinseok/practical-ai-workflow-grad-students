# 시연용 빈 파일

이 파일은 GitHub Copilot 시연 중 실시간으로 작성됩니다.

## 시연 시나리오

강사가 주석으로 컨텍스트를 제공하고, Copilot의 제안을 Tab으로 수락하며 빠르게 문서를 완성합니다.

## 시연 시작 코드

```markdown
<!--
석사 논문 연구 계획서
주제: 온라인 학습 환경에서의 학습자 참여도 분석
기간: 6개월
방법론: 혼합 연구 (설문 + 인터뷰)
-->

# 연구 계획서

## 1. 연구 제목
```

이 시점에서 Copilot이 제안을 시작합니다.

## 강의 직전 준비

- 이 파일을 빈 파일로 초기화
- 또는 새 파일 `research-plan-live.md` 생성
- Copilot 활성화 확인 (우측 하단 아이콘)

## 백업

시연 실패 시 `research-plan-completed.md` 파일 사용
