# SpecKit 실습 시연 준비 체크리스트

**시연**: SpecKit 7단계 워크플로우 소개  
**소요 시간**: 5분  
**도구**: Terminal, specify CLI, Python 3.11+  
**주의**: 선택적 도구 (시간 여유 시 시연)  

## 사전 준비 (강의 1일 전)

### 도구 설치 확인
- [ ] Python 3.11+ 설치 확인 (`python3 --version`)
- [ ] uv 패키지 매니저 설치 (선택적)
- [ ] SpecKit 설치 (`uv tool install specify-cli`)
- [ ] 설치 확인 (`specify --version`)

### 파일 준비
- [ ] `terminal-commands.md` 파일 확인
- [ ] `spec-example.md` 완성 예시 확인
- [ ] 시연용 폴더 준비 (~/Desktop/demo-speckit/)

### 설정 확인
- [ ] 터미널 글씨 크기: 16pt 이상
- [ ] 터미널에서 복사-붙여넣기 테스트
- [ ] 백업 스크린샷 준비 (명령어 실행 결과)
