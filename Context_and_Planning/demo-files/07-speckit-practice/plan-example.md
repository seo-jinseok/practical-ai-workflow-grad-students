# Plan 예시
## 온라인 학습 참여도 연구 프로젝트

### 연구 단계별 상세 계획

#### Phase 1: 문헌 조사 및 이론적 기반 구축 (Weeks 1-4)
**목표**: 온라인 학습 참여도 관련 기존 연구 파악 및 이론적 framework 개발

**주요 작업**:
- [ ] 온라인 학습 참여도 관련 핵심 문헌 수집 (50+ 편)
- [ ] 참여도 측정 도구 및 방법론 연구
- [ ] 국내외 선행연구 분석 및 비교
- [ ] 연구가치 및 연구 공백 파악
- [ ] 이론적 framework 및 가설 도출

**산출물**:
- 문헌조사 보고서 (20페이지)
- 연구문제 및 가설 구체화
- 이론적 framework 도식화

**리소스**:
- 데이터베이스: ERIC, Scopus, Google Scholar
- 검색 키워드: online learning engagement, student participation
- 예상 소요 시간: 120시간

#### Phase 2: 연구 설계 및 도구 개발 (Weeks 5-8)
**목표**: 연구 방법론 확정 및 설문 도구/인터뷰 가이드 개발

**주요 작업**:
- [ ] 연구 설계 최종 확정 (혼합 연구 방법)
- [ ] 설문지 개발 및 신뢰도·타당도 검증
- [ ] 인터뷰 가이드 개발
- [ ] IRB 신청 및 승인
- [ ] 연구 도구 파일럿 테스트
- [ ] 데이터 수집 계획 구체화

**산출물**:
- 연구계획서 (15페이지)
- 설문지 (최종버전)
- 인터뷰 가이드
- IRB 승인서

**리소스**:
- SPSS/SurveyMonkey (설문)
- NVivo (인터뷰)
- 예상 소요 시간: 100시간

#### Phase 3: 데이터 수집 (Weeks 9-16)
**목표**: 정량적·정성적 데이터 수집 완료

**정량적 데이터 수집**:
- [ ] 온라인 설문 조사 (n=200)
- [ ] 데이터 품질 검증 및 정제
- [ ] 기술통계 및 기초 분석
- [ ] 회귀분석을 위한 데이터 전처리

**정성적 데이터 수집**:
- [ ] 심층 인터뷰 진행 (n=10)
- [ ] 인터뷰 녹음 및 전사
- [ ] 질적 데이터 코딩 및 분석
- [ ] 주제별 분류 및 패턴 파악

**산출물**:
- 수집된 원시 데이터
- 데이터 분석용 데이터셋
- 인터뷰 전사본 (200페이지)

**리소스**:
- 설문 응답자 (200명)
- 인터뷰 대상자 (10명)
- 예상 소요 시간: 80시간

#### Phase 4: 데이터 분석 (Weeks 17-20)
**목표**: 정량적·정성적 데이터 통합 분석

**정량적 분석**:
- [ ] 기술통계 및 변수간 상관관계 분석
- [ ] 회귀분석을 통한 영향 요인 파악
- [ ] 가설 검증 및 결과 해석
- [ ] 데이터 시각화 및 그래프 생성

**정성적 분석**:
- [ ] 인터뷰 데이터 테마 분석
- [ ] 정량적 결과와 정성적 결과 비교분석
- [ ] 연구 현상에 대한 심층적 해석
- [ ] 통합적 결과 도출

**산출물**:
- 정량적 분석 보고서 (30페이지)
- 정성적 분석 보고서 (25페이지)
- 통합 분석 결과

**리소스**:
- SPSS (통계분석)
- NVivo (질적분석)
- R (데이터시각화)
- 예상 소요 시간: 120시간

#### Phase 5: 논문 작성 및 완성 (Weeks 21-24)
**목표**: 최종 논문 완성 및 제출

**논문 작성 단계**:
- [ ] 논문 구조 설계 및 초안 작성
- [ ] 초안 revisão 및 교정
- [ ] 지도교수 피드백 반영
- [ ] 논문 최종 완성 (학술지 투고 포맷)
- [ ] 발표 자료 준비 (학회 발표용)

**논문 구성**:
1. 서론 (Research problem, purpose, significance)
2. 이론적 배경 및 문헌고찰
3. 연구 방법론
4. 연구 결과 및 분석
5. 논의 및 제언
6. 결론 및 제한점
7. 참고문헌 및 부록

**산출물**:
- 최종 논문 (60-80페이지)
- 발표용 슬라이드
- 연구 요약

**리소스**:
- LaTeX (논문 작성)
- Google Slides (발표)
- 예상 소요 시간: 100시간

| 21-22 | Phase 5 | 논문 초안 작성 | 99% |
| 23-24 | Phase 5 | 수정 및 최종 완성 | 100% |

### 위험 관리 및 대응 전략

#### 주요 위험 요소
1. **리크루팅 실패**: 설문 응답자 부족
2. **데이터 품질**: 응답률 저하, 불완전 응답
3. **시간 지연**: 예상보다 긴 분석 시간
4. **방법론적 문제**: 통계적 유의성 미달

#### 대응 전략
- **대체 응답자 풀 확보**: 최소 2배 수 모집
- **데이터 품질 관리**: 실시간 검증 및 추적
- **전문가 자문**: 통계전문가 및 methodology consultant
- **유연한 일정 조정**: 중요한 부분 우선순위 설정

### 연구팀 구성 및 역할

**연구책임자 (석사과정 학생)**:
- 전체 연구 기획 및 실행
- 데이터 분석 및 논문 작성

**지도교수**:
- 연구 방향성 guidance
- 정기적 피드백 및 자문

**연구보조원**:
- 데이터 입력 및 정리
- 설문 배포 및 추적

**현장 전문가**:
- 인터뷰 실시 지원
- 실무적 관점 제공
