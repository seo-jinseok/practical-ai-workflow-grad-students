# Research Specification: 온라인 학습 참여도 설문 조사

**프로젝트**: 온라인 학습 환경에서의 학습자 참여도 설문 조사  
**연구자**: [이름]  
**기간**: 2주 (2025-03-01 ~ 2025-03-14)  
**유형**: 소규모 설문 조사 연구  
**버전**: 1.0  
**작성일**: 2025-03-01  

## 1. 연구 목표 (Goals)

### 주요 목표
온라인 학습 환경에서 대학생들의 학습 참여도 현황을 파악하고, 참여도에 영향을 미치는 주요 요인을 규명한다.

### 구체적 목표
1. 온라인 학습 참여도의 현재 수준 측정 (행동적, 정서적, 인지적 참여)
2. 자기조절학습 전략 사용 정도 파악
3. 참여도와 자기조절학습 전략 간 상관관계 분석
4. 학년, 전공, 온라인 학습 경험에 따른 차이 분석

### 성공 기준
- [ ] 유효 응답 100개 이상 수집
- [ ] 신뢰도 계수 (Cronbach's α) 0.7 이상
- [ ] 주요 변수 간 유의미한 상관관계 발견
- [ ] 2주 내 데이터 수집 및 분석 완료

## 2. 연구 범위 (Scope)

### 포함 사항 (In Scope)
- 대학생 (학부 1-4학년) 대상
- 온라인 학습 경험이 있는 학생
- 학습 참여도 3가지 차원 (행동적, 정서적, 인지적)
- 자기조절학습 전략 4가지 (시간 관리, 목표 설정, 자기 모니터링, 도움 요청)
- 온라인 설문 조사 (Google Forms)
- 기술통계 및 상관분석

### 제외 사항 (Out of Scope)
- 대학원생 (석박사과정)
- 온라인 학습 경험이 없는 학생
- 학습 성과 측정 (성적, 과제 점수)
- 심층 인터뷰 또는 관찰
- 실험 설계 또는 개입 연구
- 고급 통계 분석 (회귀, 구조방정식)

### 제약 조건 (Constraints)
- 시간: 2주 (데이터 수집 1주, 분석 1주)
- 예산: 없음 (무료 도구만 사용)
- 인력: 연구자 1명
- 도구: Google Forms, Python/SPSS (무료 버전)

## 3. 연구 방법 (Methods)

### 연구 설계
- 유형: 횡단 조사 연구 (Cross-sectional Survey)
- 표본: 편의 표집 (Convenience Sampling)
- 목표 표본 크기: 100-150명

### 측정 도구
#### 학습 참여도 척도
- 출처: Fredricks et al. (2004) 수정
- 문항 수: 15문항 (각 차원 5문항)
- 척도: 5점 Likert (1=전혀 그렇지 않다, 5=매우 그렇다)

#### 자기조절학습 전략 척도
- 출처: OSLQ (Barnard et al., 2009) 수정
- 문항 수: 16문항 (각 전략 4문항)
- 척도: 5점 Likert

#### 인구통계학적 정보
- 학년, 전공, 성별, 온라인 학습 경험 (학기 수)

### 데이터 수집
- 플랫폼: Google Forms
- 배포: 대학 커뮤니티, SNS
- 기간: 1주일
- 예상 응답률: 60-70%

### 데이터 분석
- 기술통계: 평균, 표준편차, 빈도
- 신뢰도 분석: Cronbach's α
- 상관분석: Pearson 상관계수
- 차이 검증: t-test, ANOVA
- 도구: Python (pandas, scipy) 또는 SPSS

## 4. 일정 (Timeline)

### Week 1: 준비 및 데이터 수집
- Day 1-2: 설문지 최종 검토 및 파일럿 테스트 (n=10)
- Day 3: 설문지 수정 및 배포
- Day 4-7: 데이터 수집 및 모니터링

### Week 2: 분석 및 보고
- Day 8-9: 데이터 정리 및 코딩
- Day 10-11: 통계 분석
- Day 12-13: 결과 해석 및 시각화
- Day 14: 보고서 작성 및 제출

### 마일스톤
- M1 (Day 3): 설문지 배포 완료
- M2 (Day 7): 100개 응답 수집 완료
- M3 (Day 11): 통계 분석 완료
- M4 (Day 14): 최종 보고서 제출

## 5. 산출물 (Deliverables)

### 주요 산출물
1. **설문지** (Google Forms)
   - 최종 설문 문항 (31문항)
   - 파일럿 테스트 결과

2. **데이터 파일**
   - 원본 데이터 (CSV)
   - 정리된 데이터 (Excel/CSV)
   - 코드북 (변수 설명)

3. **분석 결과**
   - 기술통계 표
   - 상관분석 표
   - 차이 검증 결과
   - 그래프 (막대, 산점도)

4. **최종 보고서** (10-15 페이지)
   - 서론 (연구 배경 및 목적)
   - 연구 방법 (대상, 도구, 절차, 분석)
   - 연구 결과 (기술통계, 상관, 차이)
   - 논의 및 결론 (해석, 함의, 한계)

### 품질 기준
- 신뢰도: Cronbach's α ≥ 0.7
- 유효 응답: n ≥ 100
- 결측치: < 5%
- 보고서: APA 형식 준수

## 6. 위험 요소 (Risks)

| 위험 | 가능성 | 영향 | 대응 방안 |
|------|--------|------|----------|
| 응답률 저조 (< 100개) | 중간 | 높음 | 배포 채널 확대, 기간 연장 (3일) |
| 신뢰도 낮음 (α < 0.7) | 낮음 | 중간 | 문항 제거 후 재분석 |
| 데이터 분석 지연 | 낮음 | 중간 | AI(Copilot) 활용하여 코드 생성 |
| 기술적 문제 (Google Forms) | 낮음 | 높음 | 백업 플랫폼 준비 (Naver Form) |
