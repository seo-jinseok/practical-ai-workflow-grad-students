# SpecKit 실습 시연 (연구 프로젝트 Spec 작성)

**목적**: Part 2 Section 5.3의 SpecKit 실습을 5분 시연  
**대상**: 간단한 연구 프로젝트에 SpecKit 7단계 워크플로우 적용  
**도구**: Terminal, specify CLI  
**시연 시간**: 5분  

## 📁 Files in This Demo

- `demo-scenario.md`: 시연용 연구 시나리오 (온라인 학습 설문 조사)
- `constitution-example.md`: Constitution 파일 예시
- `spec-example.md`: Spec 문서 완성 예시
- `plan-example.md`: Plan 문서 예시
- `tasks-example.md`: Tasks 문서 예시
- `terminal-commands.md`: 터미널 명령어 모음 (복사용)
- `setup-checklist.md`: 시연 전 준비 체크리스트

## 🎯 Demonstration Flow (5 minutes)

### Step 1: SpecKit 소개 및 프로젝트 초기화 (1분)
- SpecKit 7단계 워크플로우 간단 소개
- `specify init` 명령어 실행
- 생성된 폴더 구조 설명

### Step 2: Constitution 작성 (1.5분)
- Constitution 파일의 목적 설명
- 연구 원칙과 가이드라인 작성
- 간단한 연구 규칙 예시

### Step 3: Spec 문서 작성 (1.5분)
- Spec 템플릿 활용 방법
- 연구 명세서 핵심 섹션 작성
- 예시: 연구 목표, 방법, 일정

### Step 4: Plan/Tasks 미리보기 및 마무리 (1분)
- Plan 및 Tasks 파일 자동 생성 미리보기
- 대안 도구 언급 (Markdown + AI)
- "선택적 도구" 강조
