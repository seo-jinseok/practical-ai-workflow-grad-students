# 시연용 빈 파일

이 파일은 VS Code + Markdown 시연 중 실시간으로 작성됩니다.

## 시연 시나리오

강사가 이 빈 파일을 열고 다음 내용을 실시간으로 작성합니다:

1. 제목 및 구조 작성 (# ## ### 사용)
2. 연구 배경 및 질문 정리
3. 요청사항 명확히 작성
4. Preview 기능으로 결과 확인
5. AI에 복사하여 응답 받기

## 강의 직전 준비

- 이 파일의 내용을 모두 삭제하여 빈 파일로 만들기
- 또는 새 파일 `research-prompt-live.md` 생성
