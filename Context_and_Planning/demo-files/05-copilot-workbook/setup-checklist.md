## 추가 팁

### Copilot과 효과적인 협업 방법
1. **명확한 컨텍스트**: 주석으로 연구 정보를 상세히 제공
2. **구조화된 입력**: 헤딩과 목록을 사용하여 구조 명확화
3. **단계적 접근**: 복잡한 작업은 작은 단위로 나누어 요청
4. **검증의 습관**: AI 제안을 비판적으로 검토

### 학생들에게 강조할 점
1. **AI는 도구**: 연구자의 창의성과 비판적 사고가 핵심
2. **반복적 개선**: 첫 번째 결과를 바탕으로 다듬어 나가는 과정
3. **개인화**: 각 연구 주제와 전공에 맞게 템플릿 조정
4. **윤리적 사용**: AI 의존도가 과도하지 않도록 주의

## 대안 시나리오

### 시나리오 A: Copilot이 전혀 작동하지 않을 때
1. 즉시 `literature-search-completed.md` 파일 열기
2. "보통은 이렇게 Copilot이 제안합니다" 설명
3. 완성된 구조와 형식 강조
4. "다음에 직접 실습해보는 것이 중요" 안내

### 시나리오 B: 제안이 부적절할 때
1. Esc 키 눌러 제안 거부하는 동작 보이기
2. "이럴 때는 직접 입력을 하거나" 설명
3. 적절한 대안 제시
4. "AI와의 협업은 대화"라고 강조

### 시나리오 C: 시간 초과 시
1. Step 1-2만 완료 (키워드 생성까지)
2. "이런 식으로 템플릿도 생성됩니다" 완성 파일로 간단히 보여주기
3. Exercise 2-4는 자료로 참고 안내
4. 핵심 메시지 전달에 집중
