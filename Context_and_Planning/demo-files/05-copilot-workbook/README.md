# GitHub Copilot 워크북 시연 (Exercise 1: 문헌 조사 자동화)

**목적**: Part 2 Section 2.2의 Copilot 워크북 Exercise 1을 5분 시연  
**대상**: 연구 키워드 생성 및 논문 요약 템플릿 작성 시연  
**도구**: VS Code + GitHub Copilot  
**시연 시간**: 5분

- `literature-search-starter.md`: 시연용 빈 파일 (주석만 포함)

## 🎯 Demonstration Flow (5 minutes)

### Step 1: 컨텍스트 제공 (1분)
- 연구 주제를 주석으로 작성 → Copilot이 구조 제안
- 주석의 연구 정보를 바탕으로 관련 키워드 제안 예상

### Step 2: 키워드 생성 (2분)
- 한글/영문 키워드 생성 → Copilot이 관련 키워드 제안
- Tab으로 수락하며 10-15개 키워드 생성
- "## 검색 키워드" → "### 한글 키워드" → "- " 입력 시 Copilot 제안

### Step 3: 논문 요약 템플릿 생성 (1.5분)
- "## 논문 요약 템플릿" → "### 논문 1" 입력
- Copilot이 섹션 구조 제안 (기본 정보, 연구 목적, 연구 방법, 주요 결과, 관련성)
- 4가지 핵심 섹션 자동 완성

### Step 4: 마무리 및 다음 단계 (0.5분)
- Exercise 2-4 자료 안내
- Part 2 Copilot 워크북 내용 소개
