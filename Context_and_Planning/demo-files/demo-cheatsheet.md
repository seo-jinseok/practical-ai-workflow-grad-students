## 시연 1: VS Code + Markdown

**타이밍**: Part 1, 강의 시작 후 약 25분
**소요 시간**: 5분
**파일**: `demo-files/01-vscode-markdown/research-prompt.md`

### 빠른 체크
- [ ] VS Code 실행
- [ ] 빈 파일 열림
- [ ] 글씨 크기 18pt 이상
- [ ] Preview 단축키: `Cmd/Ctrl + K V`
- [ ] 백업: `research-prompt-completed.md`

### 핵심 멘트
1. "구조화된 입력이 구조화된 출력을 만듭니다"
2. "Markdown은 5분이면 배울 수 있습니다"
3. "AI가 이런 구조를 더 잘 이해합니다"

### 시연 순서
1. 제목 작성 (# ## ###)
2. 내용 정리 (목록, 번호)
3. Preview 확인
4. AI에 복사
5. 응답 저장
