### Step 1: Context Engineering (1분)

#### 폴더 구조 생성 (30초)

**강사 액션**:
1. `online-learning-study/` 폴더 생성
2. Markdown 파일 생성: `연구컨텍스트.md`, `AGENTS.md`
3. 기본 구조 입력

**강사 시연 코드**:
```
# 연구컨텍스트.md
## 연구 주제
온라인 학습 환경에서 학습자 참여도 영향 요인 분석

## 연구 목적
- 온라인 학습 참여도 수준 파악
- 영향 요인 분석
- 참여도 향상 전략 개발

# AGENTS.md  
## AI 협업 규칙
- AI는 정보 수집 및 초안 작성
- 인간은 검증 및 최종 결정
- 모든 결과는 검증 체크리스트 통과
```

**강사 멘트**: "VS Code + Markdown + AI로 시작합니다. Part 1의 Context Engineering입니다."

#### Spec-driven Planning 연계 (30초)

**강사 액션**:
1. Spec 템플릿 파일 생성: `연구명세서.md`
2. 기본 구조 입력

**강사 시연 코드**:
```
# 연구명세서.md
## 연구 목표
온라인 학습 참여도 향상을 위한 요인 파악

## 연구 방법
혼합 연구 (설문 n=200 + 인터뷰 n=10)

## 일정
6개월 (24주)

## 기대 결과
참여도 영향 요인 파악 및 개선 전략 도출
```

**강사 멘트**: "이제 Part 2의 Spec-driven Planning과 연결됩니다."
