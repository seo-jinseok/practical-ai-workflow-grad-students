# Week 8: 문헌 조사 결과 시연
## Copilot 워크북 + 2025 도구 통합

### 시연 개요 (30초)

**강사 멘트**: "Week 8, 문헌 조사 단계의 결과를 보여드리겠습니다.
Part 2의 Copilot 워크북과 Part 3의 2025 도구가 어떻게 통합되는지 확인해보겠습니다."

**강의실 설정**:
- 시연 파일: `05-copilot-workbook/literature-search-completed.md`
- Elicit, NotebookLM 결과 준비
- 타이머: 1.5분

### Step 1: Copilot 워크북 결과 (45초)

#### 키워드 생성 완료 (15초)

**강사 액션**:
1. `literature-search-completed.md` 파일 열기
2. 검색 키워드 섹션 표시

**강의실 화면 표시**:
```markdown
## 검색 키워드

### 한글 키워드
- 온라인 학습
- 학습자 참여도
- 원격 교육
- 이러닝
- 학습 동기
- 자기조절학습

### English Keywords
- online learning
- learner engagement
- student participation
- e-learning
- learning motivation
- self-regulated learning
```

**강사 멘트**: "Copilot 워크북 Exercise 1로 생성한 키워드입니다. 주석으로 컨텍스트를 제공하니 Copilot이 적절한 제안을 했습니다."

#### 논문 요약 템플릿 (30초)

**강사 액션**:
1. 논문 요약 템플릿 섹션으로 이동
2. 완성된 구조 표시

**강의실 화면 표시**:
```markdown
## 논문 요약 템플릿

### 논문 1
**기본 정보**
- 제목: [논문 제목]
- 저자: [저자명]
- 연도: [연도]
- 저널: [저널명]

**연구 목적**
[연구 목적 설명]

**연구 방법**
[방법론 설명]

**주요 결과**
[핵심 결과]

**본 연구와의 관련성**
[관련성 분석]
```

**강사 멘트**: "Copilot이 제안한 표준화된 템플릿입니다. 모든 논문을 일관된 형식으로 정리할 수 있습니다."

### Step 2: 2025 도구 통합 결과 (45초)

#### Elicit Systematic Review (20초)

**강사 멘트**: "Elicit을 사용해서 체계적 문헌 검토를 수행했습니다."

**강의실 화면 표시**:
- Elicit 인터페이스 스크린샷
- 15개 핵심 논문 목록
- 연구 질문별 분류 결과

**강의실 화면 텍스트**:
```
Elicit 검색 결과:
- 온라인 학습 참여도: 15편
- 자기조절학습: 12편  
- 온라인 상호작용: 8편
- 학습 성과: 10편

주요 발견:
- 참여도는 동기, 자기조절, 상호작용과 밀접한 상관관계
- 5년 내 최신 연구 80% 확보
```

#### NotebookLM Mind Map (25초)

**강사 멘트**: "NotebookLM으로 연구 지식지도와 오디오 요약을 만들었습니다."

**강의실 화면 표시**:
- Mind Map 구조
- 노드별 연결 관계
- 오디오 요약 링크

**강의실 화면 텍스트**:
```
NotebookLM 결과물:
├── 온라인 학습 참여도
│   ├── 정의 및 개념
│   ├── 측정 도구
│   └── 영향 요인
├── 이론적 framework
│   ├── 자기조절학습 이론
│   ├── 참여도 모델
│   └── 통합 framework
└── 선행연구 분석
    ├── 정량적 연구
    ├── 정성적 연구
    └── 혼합 연구
```

**강사 멘트**: "15개 논문을 종합하여 시각적 지식지도를 생성했습니다. 핵심 개념과 연결 관계를 한눈에 파악할 수 있습니다."

### Step 3: 통합 워크플로우 효과 (15초)

#### 도구별 강점 활용 (15초)

**강사 멘트**: "세 도구가 각기 다른 강점을 보여줍니다:
- Copilot: 구조화된 템플릿과 키워드 생성
- Elicit: 체계적 문헌 검색과 분류
- NotebookLM: 지식 통합과 시각화

이것이 Part 1-2-3 통합 워크플로우입니다."

### 시연 중 점검 질문

**강사 멘트**: "여기서 확인해보겠습니다:'

- **주석 컨텍스트 제공의 효과**: 'Copilot이 더 정확한 제안을 했습니까?'
- **다중 도구 활용**: '각 도구가 다른 가치를 제공했습니까?'
- **시간 절약 효과**: '직접 조사 대비 얼마나 효율적이었습니까?'
