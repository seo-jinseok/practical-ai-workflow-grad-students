# 강의 전체 구조 문서 생성

## 1. 시간 배분 계획 (총 120분)

### 도입부 (15분)
- 강의 목표 및 개요 소개 (3분)
- 생성형 AI의 현황과 대학원 연구/학습에서의 활용 가능성 (7분)
- 청중 배경 파악 및 기대사항 확인 (3분)
- 강의 로드맵 제시 (2분)

### Part 1: Context Engineering (40분)
- 섹션 소개 및 학습 목표 (3분)
- 효과적인 프롬프트 작성 원칙 (12분)
  - 명확성과 구체성
  - 구조화된 입력 (markdown 활용)
  - 예시 기반 학습 (few-shot prompting)
- 컨텍스트 관리 전략 (15분)
  - Markdown을 활용한 구조화
  - MCP(Model Context Protocol) 개념 소개
  - 다양한 MCP 도구 소개 (byterover, sequential thinking, context7, brave/duckduckgo/tavily search, task-master-mcp, perplexity, git)
- 대학원 연구 사례 적용 (8분)
  - 논문 작성 지원
  - 문헌 조사 및 정리
  - 혁신보고서 작성 사례
- 실습 예제 및 토론 (2분)

### Part 2: Planning & Task Management (40분)
- 섹션 소개 및 학습 목표 (3분)
- Spec-driven Planning 개념 (12분)
  - 명세 기반 계획의 중요성
  - AI를 활용한 요구사항 분석
  - 구조화된 계획 문서 작성법
- AI 기반 작업 분해 및 관리 (15분)
  - 대규모 프로젝트를 작은 태스크로 분해
  - 우선순위 설정 및 일정 관리
  - 진행 상황 추적 방법론
- Review와 Verification 프로세스 (8분)
  - AI 출력물 검증 전략
  - 품질 관리 체크리스트
  - 반복적 개선 프로세스
- 대학원 프로젝트 적용 사례 (2분)

### Part 3: 실전 워크플로우 통합 (20분)
- 섹션 소개 (2분)
- End-to-End 워크플로우 시연 (12분)
  - Context Engineering + Planning 통합 시나리오
  - 실제 대학원 연구 프로젝트 예시 (논문 작성 또는 데이터 분석)
  - 도구 시연 통합 (VS Code, GitHub Copilot, GitHub Projects, task-master-mcp)
- 다양한 전공별 적용 방안 (4분)
  - 인문사회계열 사례
  - 이공계열 사례
  - 예체능계열 사례
- 질의응답 및 토론 (2분)

### 마무리 (5분)
- 핵심 내용 요약 (2분)
- 실천 가이드 및 다음 단계 제안 (2분)
- 추가 학습 리소스 안내 (1분)

## 2. 각 섹션별 학습 목표

### 도입부 학습 목표
- 생성형 AI의 현재 수준과 한계를 이해한다
- 대학원 연구/학습에서 AI 활용의 실질적 가치를 인식한다
- 강의 전체 흐름과 기대 성과를 파악한다

### Part 1 학습 목표
- 효과적인 프롬프트 작성 원칙을 이해하고 적용할 수 있다
- Markdown을 활용한 구조화된 입력 방법을 습득한다
- MCP 개념과 다양한 컨텍스트 관리 도구를 이해한다
- 자신의 연구 분야에 Context Engineering을 적용할 수 있다

### Part 2 학습 목표
- Spec-driven planning의 개념과 장점을 이해한다
- AI를 활용한 작업 분해 및 관리 방법론을 습득한다
- AI 출력물의 검증 및 품질 관리 전략을 수립할 수 있다
- 대학원 프로젝트에 체계적인 계획 및 관리 기법을 적용할 수 있다

### Part 3 학습 목표
- Context Engineering과 Planning을 통합한 워크플로우를 이해한다
- 실제 연구 시나리오에서 end-to-end 프로세스를 적용할 수 있다
- 자신의 전공 분야에 맞는 AI 활용 전략을 수립할 수 있다
- 실용적인 도구들을 활용한 생산성 향상 방법을 습득한다

### 마무리 학습 목표
- 강의 핵심 내용을 체계적으로 정리한다
- 즉시 실천 가능한 액션 아이템을 도출한다
- 지속적인 학습을 위한 리소스를 확보한다

## 3. 도구 시연 통합 계획

### VS Code 시연 (Part 1 중반, 5분)
- Markdown 기반 구조화된 프롬프트 작성 시연
- 확장 기능을 활용한 효율적인 문서 작성

### GitHub Copilot 시연 (Part 2 중반, 5분)
- 코드/문서 자동 완성 기능
- 주석 기반 코드 생성 (spec-driven 접근 예시)

### GitHub Projects 시연 (Part 2 후반, 5분)
- 명세 기반 프로젝트 계획 수립
- 요구사항 문서화 및 추적

### task-master-mcp 시연 (Part 3 중반, 5분)
- 작업 분해 및 관리 자동화
- Context Engineering + Planning 통합 워크플로우

## 4. 청중 맞춤 전략

### 부산 소재 사립대 대학원생 특성 고려
- 다양한 전공 배경: 범용적이고 전공 중립적인 예시 사용
- 중간 수준: 기초 개념부터 시작하되 실용적 활용에 집중
- 실무 지향: 즉시 적용 가능한 실전 예제 중심
- 지역 특성: 부산/경남 지역 사례나 맥락 활용 가능

### 동기부여 요소
- 연구 생산성 향상 사례
- 논문 작성 시간 단축 경험담
- 취업/진로에서의 AI 활용 역량 중요성
- 동료 연구자들의 성공 사례
