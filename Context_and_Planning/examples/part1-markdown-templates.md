```markdown
# 문헌 비교 분석

## 비교할 논문들
1. [저자1, 연도]: [제목]
2. [저자2, 연도]: [제목]
3. [저자3, 연도]: [제목]

## 비교 관점
다음 관점에서 비교 분석해줘:

1. **연구 방법**
   - 공통점
   - 차이점

2. **주요 발견**
   - 일치하는 결과
   - 상반되는 결과

3. **이론적 틀**
   - 사용된 이론
   - 이론 적용 방식

4. **한계점**
   - 공통 한계
   - 개별 한계

5. **연구 갭**
   - 이 논문들이 다루지 못한 부분

결과를 표 형식으로 정리해줘.
```

## 📊 데이터 분석

### 분석 방법 선택
```markdown
# 적절한 분석 방법 추천

## 연구 상황
- **연구 질문**: [질문]
- **독립 변수**: [변수명] (유형: [명목/서열/등간/비율])
- **종속 변수**: [변수명] (유형: [명목/서열/등간/비율])
- **표본 크기**: n = [숫자]
- **데이터 분포**: [정규/비정규/모름]

## 요청사항
1. 적절한 통계 분석 방법 추천
2. 각 방법의 장단점
3. 가정 검증 방법
4. 분석 순서
5. 결과 해석 방법

초보자도 이해할 수 있게 설명해줘.
```

### 결과 해석

```markdown
# 통계 결과 해석 도움

## 분석 결과
아래에 결과를 붙여넣으세요 (펜스 필요 없음):
~~~~
[통계 출력 결과 붙여넣기]
~~~~

## 요청사항
위 결과를 다음 관점에서 해석해줘:

1. **통계적 유의성**
   - p-value 해석
   - 효과 크기

2. **실질적 의미**
   - 결과가 의미하는 바
   - 실제 적용 가능성

3. **논문 작성용 문장**
   - Results 섹션에 쓸 문장
   - APA 스타일

4. **시각화 제안**
   - 적절한 그래프 유형
   - 그래프 작성 방법
```

```markdown
6. 마무리 take-home message
```

## 사용 가이드

### 템플릿 활용 방법
1. **복사**: 필요한 템플릿을 복사
2. **수정**: [대괄호] 부분을 자신의 정보로 교체
3. **저장**: `.md` 파일로 저장
4. **재사용**: 프로젝트 전반에 걸쳐 반복 사용
