# Part 2 작업 분해 예제 모음

이 문서는 다양한 유형의 프로젝트를 작업 분해한 실제 예제를 제공합니다.

## 예제 1: 석사 논문 작업 분해

### 프로젝트 개요
- **주제**: 온라인 학습 환경에서의 자기조절학습 전략
- **기간**: 6개월
- **방법**: 혼합 연구 (설문 + 인터뷰)

### AI 프롬프트
```markdown
# 프로젝트
석사 논문: 온라인 학습 환경에서의 자기조절학습 전략

## 기간
6개월 (2025년 3월 - 8월)

## 방법
- 설문 조사 (n=200)
- 심층 인터뷰 (n=10)

## 요청
이 프로젝트를 완료 가능한 작업들로 분해해줘.
각 작업은 1-2주 내 완료 가능한 크기로.

각 작업에 포함:
- 작업명
- 예상 소요 시간
- 선행 작업
- 산출물
- 우선순위
```

## 예제 2: 데이터 분석 프로젝트 작업 분해

### 프로젝트: 설문 데이터 분석

```markdown
# 작업 분해: 설문 데이터 분석 프로젝트

## 프로젝트 정보
- 데이터: 설문 응답 200건
- 변수: 10개
- 목표: 학습 전략과 성취도의 관계 분석
- 기간: 3주

## 작업 목록

### Week 1: 데이터 준비 및 탐색

**작업 1.1: 데이터 로딩 및 구조 파악** (2시간)
- 데이터 파일 읽기
- 변수 확인 및 데이터 타입 점검
- 기본 통계 확인
```python
import pandas as pd
df = pd.read_csv('survey_data.csv')
df.info()
df.describe()
```

**작업 1.2: 결측치 분석 및 처리** (4시간)
- 결측치 패턴 분석
- 처리 방법 결정 (삭제/대체)
- 처리 실행 및 검증

**작업 1.3: 이상치 탐지 및 처리** (3시간)
- 박스플롯, Z-score로 이상치 탐지
- 이상치 처리 방법 결정
- 처리 실행

**작업 1.4: 변수 변환** (3시간)
- 역코딩 필요 변수 처리
- 합산 점수 계산
- 더미 변수 생성

**작업 1.5: 탐색적 데이터 분석 (EDA)** (6시간)
- 변수별 분포 확인
- 상관관계 분석
- 시각화 (히스토그램, 산점도)
- EDA 보고서 작성

### Week 2: 통계 분석

**작업 2.1: 가정 검증** (3시간)
- 정규성 검정
- 등분산성 검정
- 독립성 검정
- 다중공선성 검정

**작업 2.2: 기술 통계 및 상관 분석** (4시간)
- 평균, 표준편차 계산
- 상관관계 매트릭스
- 결과 표 작성

**작업 2.3: 집단 간 차이 검증** (4시간)
- t-검정 또는 ANOVA
- 사후 검정 (필요시)
- 효과 크기 계산

**작업 2.4: 회귀 분석** (6시간)
- 단순 회귀 분석
- 다중 회귀 분석
- 모델 적합도 평가
- 결과 해석

**작업 2.5: 추가 분석** (4시간)
- 매개 효과 분석 (선택적)
- 조절 효과 분석 (선택적)
- 하위 집단 분석

### Week 3: 결과 정리 및 작성

**작업 3.1: 결과 표 작성** (4시간)
- Table 1: 기술 통계 및 상관관계
- Table 2: 집단 간 차이 검증
- Table 3: 회귀 분석 결과
- APA 형식 준수

**작업 3.2: 결과 그래프 작성** (6시간)
- Figure 1: 연구 모형
- Figure 2-3: 주요 결과 시각화
- Figure 4: 상호작용 효과 (필요시)
- 고품질 그래프 (300 dpi 이상)

**작업 3.3: Results 섹션 작성** (8시간)
- 기술 통계 서술
- 가설 검증 결과 서술
- 표와 그래프 설명
- AI 활용: 초안 작성

**작업 3.4: 결과 검토 및 수정** (4시간)
- 통계 결과 재확인
- 동료 검토
- 지도교수 피드백 반영

**작업 3.5: 최종 정리** (2시간)
- 모든 파일 정리
- 분석 스크립트 문서화
- 백업 및 아카이빙
```

## 작업 분해 원칙

### SMART 작업 정의
- **Specific**: "논문 쓰기" ❌ → "서론 초안 작성" ✅
- **Measurable**: "열심히 하기" ❌ → "논문 10편 읽기" ✅
- **Achievable**: "1주일에 논문 완성" ❌ → "1주일에 서론 완성" ✅
- **Relevant**: 프로젝트 목표와 직접 연관
- **Time-bound**: 명확한 마감일 설정

### 적절한 작업 크기
- **너무 큼**: "데이터 분석" (2주 이상)
- **적절함**: "기술 통계 분석" (3-4시간)
- **너무 작음**: "평균 계산" (10분)

**권장 크기**: 2시간 - 2주
- 2시간 미만: 다른 작업과 통합
- 2주 초과: 더 작은 단위로 분해

### 의존성 표시 방법

**방법 1: 텍스트**
```
작업 5: 데이터 분석
- 선행 작업: 작업 3 (데이터 수집), 작업 4 (데이터 정제)
```

**방법 2: 다이어그램**
```
작업 1 → 작업 2 → 작업 4
           ↓
        작업 3 → 작업 5
```

**방법 3: 표**
| 작업 | 선행 작업 | 후속 작업 |
|------|----------|----------|
| 작업 1 | - | 작업 2 |
| 작업 2 | 작업 1 | 작업 3, 4 |

## 요청사항
이 프로젝트를 완료하기 위한 상세 작업 목록을 만들어줘.

각 작업은:
- 1-2주 내 완료 가능한 크기
- 명확한 산출물
- 예상 소요 시간 (시간 단위)
- 선행 작업 (의존성)
- 우선순위 (High/Medium/Low)

작업을 주차별로 그룹화하고,
병렬로 진행 가능한 작업은 표시해줘.
