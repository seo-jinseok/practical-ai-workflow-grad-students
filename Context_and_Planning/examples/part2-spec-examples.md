```markdown
# 석사 논문 프로젝트 명세서

## 📋 기본 정보
- **프로젝트명**: 청년 세대의 직업 가치관 변화 연구
- **연구자**: [이름]
- **지도교수**: [교수명]
- **기간**: 2025년 3월 - 2025년 12월 (10개월)
- **유형**: 질적 연구 (심층 인터뷰)

## 🎯 연구 목표
청년 세대(20-30대)의 직업 선택 기준과 가치관을 탐색하고,
이전 세대와의 차이점 및 형성 배경을 분석한다.

## 🔍 연구 질문
1. 청년들이 직업을 선택할 때 가장 중요하게 생각하는 가치는 무엇인가?
2. 이러한 가치관이 형성된 사회적 배경은 무엇인가?
3. 세대 간 직업 가치관의 차이는 어떻게 나타나는가?

## 📚 연구 방법
### 참여자
- 대상: 20-30대 직장인 및 구직자
- 표본: 15-20명 (목적 표집)
- 선정 기준: 다양한 직업군, 경력 단계

### 데이터 수집
- 방법: 반구조화된 심층 인터뷰
- 시간: 1인당 60-90분
- 장소: 참여자 선호 장소 또는 온라인
- 녹음: 동의 하에 녹음 및 전사

### 데이터 분석
- 방법: 주제 분석 (Thematic Analysis)
- 도구: NVivo 또는 수동 코딩
- 절차: 개방 코딩 → 축 코딩 → 선택 코딩

## 📅 일정 계획
| 단계 | 작업 | 기간 | 산출물 |
|------|------|------|--------|
| 1 | 문헌 고찰 | 3-4월 (2개월) | 문헌 고찰 초안 |
| 2 | 연구 설계 | 4월 (1개월) | 연구 계획서, IRB 신청 |
| 3 | IRB 승인 | 5월 (1개월) | IRB 승인서 |
| 4 | 참여자 모집 | 5-6월 (1개월) | 참여자 15-20명 |
| 5 | 인터뷰 실시 | 6-8월 (3개월) | 녹음 파일, 전사본 |
| 6 | 데이터 분석 | 8-10월 (3개월) | 코딩 결과, 주제 |
| 7 | 논문 작성 | 10-12월 (3개월) | 논문 초안 |
| 8 | 수정 및 제출 | 12월 (1개월) | 최종 논문 |

## 📦 주요 산출물
```

### 예제 2: 교육학 - 실험 연구 프로젝트

```markdown
# 박사 논문 프로젝트 명세서

## 📋 기본 정보
- **프로젝트명**: 게이미피케이션이 학습 동기에 미치는 영향
- **연구 유형**: 준실험 연구 (Quasi-experimental)
- **기간**: 2025년 3월 - 2026년 2월 (12개월)

## 🎯 연구 목표
게이미피케이션 요소가 대학생의 학습 동기와 학업 성취도에
미치는 영향을 실증적으로 검증한다.

## 🔬 연구 설계
### 설계 유형
- 준실험 설계 (Nonequivalent Control Group Design)
- 실험집단 vs 통제집단
- 사전-사후 검사

### 참여자
- 대상: 대학교 1-2학년생
- 표본: 실험집단 60명, 통제집단 60명 (총 120명)
- 모집: [대학명] 교양 과목 수강생

### 독립변수
- 게이미피케이션 요소 (포인트, 배지, 리더보드, 레벨)

### 종속변수
- 학습 동기 (MSLQ 척도)
- 학업 성취도 (중간/기말 시험 점수)
- 학습 참여도 (LMS 로그 데이터)

### 통제변수
- 사전 학업 성취도
- 학습 스타일
- 컴퓨터 활용 능력

## 📅 연구 일정
| 단계 | 기간 | 주요 작업 |
|------|------|----------|
| 준비 | 3-4월 | 문헌 고찰, IRB 신청, 도구 개발 |
| 사전 검사 | 5월 1주 | 사전 설문, 사전 시험 |
| 실험 처치 | 5-7월 | 게이미피케이션 적용 (12주) |
| 사후 검사 | 7월 4주 | 사후 설문, 사후 시험 |
| 데이터 분석 | 8-10월 | 통계 분석, 결과 해석 |
| 논문 작성 | 11월-2월 | 초안 작성, 수정, 제출 |

## 📊 데이터 분석 계획
### 기술 통계
- 평균, 표준편차, 빈도 분석

### 추론 통계
- 독립표본 t-검정 (집단 간 차이)
- 대응표본 t-검정 (사전-사후 차이)
- 이원 반복측정 ANOVA (집단 x 시점)
- 다중 회귀 분석 (예측 변수)

### 분석 도구
- SPSS 28.0
- Python (pandas, scipy, statsmodels)

## 📦 산출물 체크리스트
- [ ] 연구 계획서
- [ ] IRB 승인서
- [ ] 게이미피케이션 시스템 설계
- [ ] 측정 도구 (설문지)
- [ ] 사전 검사 데이터
- [ ] 사후 검사 데이터
- [ ] LMS 로그 데이터
- [ ] 통계 분석 결과
- [ ] 논문 각 장 초안
- [ ] 최종 논문

## 🎯 성공 기준
- 참여자 유지율 80% 이상
- 통계적 유의미한 결과 도출
- 학술지 게재 가능 수준의 논문
- 예정된 일정 내 완료
```

```markdown
# 석사 논문 프로젝트: 단백질 발현 연구

## 📋 기본 정보
- **연구 주제**: A 단백질의 발현 조절 메커니즘 규명
- **연구 유형**: 실험 연구 (분자생물학)
- **기간**: 2025년 3월 - 2026년 2월 (12개월)

## 🎯 연구 목표
X 조건에서 A 단백질의 발현이 조절되는 메커니즘을
분자 수준에서 규명하고, 관련 신호전달 경로를 밝힌다.

## 🔬 실험 설계
### 실험 1: 단백질 발현 확인
- 방법: Western Blot
- 세포주: HEK293, HeLa, MCF-7
- 처리: 약물 A (0, 10, 50, 100 μM)
- 시간: 0, 6, 12, 24시간
- 반복: 3회 독립 실험

### 실험 2: mRNA 발현 분석
- 방법: qRT-PCR
- 유전자: A, B, C (관련 유전자)
- 조건: 실험 1과 동일
- 반복: 3회 독립 실험

### 실험 3: 신호전달 경로 분석
- 방법: Immunoprecipitation, Mass Spectrometry
- 목표: 상호작용 단백질 동정

### 실험 4: 기능 검증
- 방법: siRNA knockdown, Overexpression
- 측정: 세포 증식, 이동, 사멸

## 📅 실험 일정
| 월 | 주요 실험 | 예상 결과 |
|----|----------|----------|
| 3월 | 실험 1 (Western Blot) | 발현 패턴 확인 |
| 4월 | 실험 2 (qRT-PCR) | mRNA 수준 확인 |
| 5-6월 | 실험 3 (IP-MS) | 상호작용 단백질 |
| 7-8월 | 실험 4 (기능 검증) | 기능적 역할 |
| 9월 | 추가 실험 (필요시) | 보완 데이터 |
| 10-2월 | 논문 작성 | 최종 논문 |

## 🧪 재료 및 시약
### 세포주
- [ ] HEK293 (ATCC)
- [ ] HeLa (ATCC)
- [ ] MCF-7 (ATCC)

### 주요 시약

### 장비

## 📊 데이터 분석
### 통계 분석

- Two-way ANOVA (세포주 x 농도)
```

```markdown
## 📚 연구 범위
### 이론 연구
```

## 사용 가이드

### Spec 문서 작성 프로세스

1. **초안 작성** (30분-1시간)
   - 위 템플릿 복사
   - [대괄호] 부분을 자신의 정보로 채우기
   - 불필요한 섹션 삭제, 필요한 섹션 추가

2. **AI와 정교화** (1-2시간)
   - AI에게 Spec 초안 제공
   - "이 Spec을 검토하고 누락된 부분을 제안해줘"
   - "각 작업의 예상 소요 시간을 제안해줘"
   - "위험 요소를 추가로 제안해줘"

3. **검토 및 수정** (30분)
   - 지도교수 또는 동료에게 공유
   - 피드백 반영
   - 최종 버전 확정

4. **지속적 업데이트**
   - 프로젝트 진행 중 변경사항 반영
   - 버전 관리 (v1.0, v1.1, v2.0 등)
   - 완료된 항목 체크
