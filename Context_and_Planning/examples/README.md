## 💡 사용 가이드

### 강사용 가이드

#### 준비 사항
1. 모든 예제 파일을 미리 읽어보기
2. 자신의 전공/경험에 맞는 예제 선택
3. 필요시 예제 수정 또는 추가
4. VS Code에서 예제 파일 열어두기

#### 시연 팁
1. **천천히**: 각 섹션을 천천히 설명
2. **강조**: 중요한 부분은 하이라이트
3. **비교**: Before/After를 명확히 대비
4. **질문**: "여러분의 전공에서는?" 등 참여 유도

### 학생용 가이드

#### 파일 다운로드
```bash
# Git 사용 시
git clone [저장소 URL]
cd examples

# 또는 개별 파일 다운로드
# GitHub에서 Raw 버튼 클릭 → 저장
```

#### 사용 방법
1. **복사**: 필요한 템플릿/예제 복사
2. **수정**: [대괄호] 부분을 자신의 정보로 교체
3. **저장**: 자신의 프로젝트 폴더에 저장
4. **활용**: 연구 프로젝트에 적용

#### 맞춤화
- 자신의 전공에 맞게 용어 수정
- 필요한 섹션 추가/삭제
- 지도교수 요구사항 반영
