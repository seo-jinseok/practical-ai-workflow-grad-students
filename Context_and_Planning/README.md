# 생성형 AI 특강 - 대학원 연구자를 위한 AI 활용법

이 저장소는 생성형 AI를 대학원 연구와 학습에 효과적으로 활용하는 방법을 다루는 특강의 계획 문서들을 포함합니다.

## 📋 강의 개요

이 특강은 생성형 AI의 현재 상황을 이해하고, 대학원 연구자들이 AI를 효과적으로 활용할 수 있도록 돕는 것을 목표로 합니다. 특히 Context Engineering과 Planning을 통해 AI를 진정한 연구 파트너로 만드는 방법을 중점적으로 다룹니다.

## 🎓 학습 목표

### Part 1: Context Engineering
- 효과적인 프롬프트 작성 원칙을 이해하고 적용할 수 있다
- Markdown을 활용한 구조화된 입력 방법을 습득한다
- MCP 개념과 다양한 컨텍스트 관리 도구를 이해한다

### Part 2: Planning & Task Management
- Spec-driven planning의 개념과 장점을 이해한다
- AI를 활용한 작업 분해 및 관리 방법론을 습득한다
- AI 출력물의 검증 및 품질 관리 전략을 수립할 수 있다

### Part 3: 실전 워크플로우 통합
- Context Engineering과 Planning을 통합한 워크플로우를 이해한다
- 실제 연구 시나리오에서 end-to-end 프로세스를 적용할 수 있다
