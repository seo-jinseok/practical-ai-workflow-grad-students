**다른 시나리오 예시** (14pt):

**Part 1 기법 활용** (16pt):

✓ 5W1H 원칙 적용
✓ 명확한 연구 질문 정의

**실제 작성 예시** (14pt, 고정폭 폰트):
```markdown
# 연구 프로젝트: 온라인 학습 환경 분석

## 연구 배경
- 코로나19 이후 온라인 학습 급증
- 자기조절학습 능력의 중요성 증가

## 연구 질문
1. 온라인 학습 환경에서 학습자들은 어떤
   자기조절학습 전략을 사용하는가?
2. 효과적인 전략과 비효과적인 전략의
   차이는 무엇인가?

## 연구 방법
- 설문 조사 (n=200)
- 심층 인터뷰 (n=10)
- 기간: 6개월
```

**AI 활용** (16pt):
→ 이 컨텍스트를 AI에게 제공하고 문헌 조사 방향 제안 받기

## 슬라이드 9: Step 3 - 실행 및 관리
**타입**: 프로세스 슬라이드
**소요 시간**: 1분 30초

### 콘텐츠
**제목**: "Step 3: 실행 및 진행 관리"

**작업 목표** (16pt):
계획된 작업을 실행하고 AI와 지속적으로 소통하며 진행

**실행 프로세스** (16pt):

**매일/매주 루틴**:
1. **오늘/이번 주 작업 확인**
   - AI: "다음 우선순위 작업은 무엇인가요?"

2. **작업 실행**
   - 문헌 검토, 데이터 수집, 분석, 작성 등
   - AI 도움: 초안 작성, 코드 생성, 분석 지원

3. **진행 상황 기록**
   - 완료한 작업 체크
   - AI에게 진행 상황 보고

4. **다음 단계 확인**
   - AI가 다음 작업 제안
   - 일정 조정 필요 여부 확인

**AI 활용 예시** (14pt):
```markdown
오늘 다음 작업을 완료했습니다:
- [x] 논문 5편 검토 및 요약
- [x] 설문 도구 초안 작성

진행 상황을 업데이트하고 다음 할 일을 알려줘.
```

**📁 자료 열기**: [워크플로우 시나리오 예시](./examples/part3-workflow-scenarios.md)

| **Month 3-4** | 데이터 수집 | 8주 | 참여자 모집 문구, 리마인더 자동화 |
| **Month 5** | 데이터 분석 | 4주 | 분석 코드(또는 오픈소스 LLM 코드 어시스트) |
| **Month 6** | 논문 작성 & 제출 | 4주 | 각 섹션 초안, 교정 지원 |

**주간 루틴** (16pt):
- **월요일**: AI에게 이번 주 우선순위 작업 확인
- **수-금**: 작업 실행, 막힌 부분 AI와 브레인스토밍
- **금요일**: 주간 진행 보고, AI에게 다음 주 계획 제안 받기

**핵심 포인트** (16pt, 강조색):
✓ 매 단계마다 AI와 대화
✓ 지속적인 검증과 개선
✓ 유연한 일정 조정

**📁 자료 열기**: [워크플로우 시나리오 예시](./examples/part3-workflow-scenarios.md)

### 시각 요소

## 슬라이드 12: 전공별 적용 - 인문사회 vs 이공계열
**타입**: 예시 슬라이드
**소요 시간**: 1분 15초

### 콘텐츠
**제목**: "전공별 적용: 연구 유형에 따른 활용법"

**인문사회계열 적용** (16pt):
**시나리오**: 사회학 석사 - 청년 직업 가치관 변화 연구

**워크플로우** (14pt):
- **Context**: 인터뷰 가이드 구조화, 이론적 배경 정의
- **Planning**: 인터뷰 → 전사 → 코딩 → 분석 → 작성
- **실행**: AI 코딩 제안 + 연구자 정교화
- **검증**: 연구자 재검토 + 동료 교차 검증

**핵심 팁** (14pt, 강조색):
💡 AI는 초기 패턴 발견에 강점, 최종 해석은 연구자의 몫

**이공계열 적용** (16pt):
**시나리오**: 컴퓨터공학 석사 - 머신러닝 추천 시스템 개발

**워크플로우** (14pt):
- **Context**: 연구 목표, 데이터셋, 평가 지표 정의
- **Planning**: 데이터 전처리 → 모델 구현 → 실험 → 논문 작성
- **실행**: 코드 작성 (Copilot) + 결과 해석 (AI 대화)
- **검증**: 코드 리뷰 + 실험 재현성 확인

**핵심 팁** (14pt, 강조색):
💡 코드 작성은 AI, 결과 해석은 AI 대화, 검증은 사람

**공통 원칙** (16pt, 강조색):
✓ 모든 전공에서 동일한 4단계 워크플로우 적용
✓ AI는 보조 도구, 연구자의 전문성 판단은 필수
✓ 도구는 달라도 원칙은 동일

### 시각 요소
- 인문사회 vs 이공계를 좌우로 배치
- 4단계 플로우차트를 간단히
- 핵심 팁을 색상 구분

### 발표자 노트
- "모든 전공이 동일한 워크플로우 원칙을 따릅니다"
- "활용 방법만 전공 특성에 맞게 조정하면 됩니다"
- "예체능도 이론 연구와 논문 작성에서 동일하게 활용할 수 있습니다"
