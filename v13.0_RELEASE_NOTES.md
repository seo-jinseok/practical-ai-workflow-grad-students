# v13.0 Release Notes
## 대학원생을 위한 실용적 AI 워크플로우

**릴리스 버전**: v13.0  
**릴리스 날짜**: 2025-11-15 (예정)  
**이전 버전**: v12.0 (2025-11-01)  
**주요 변경**: 3부작 구조, 2025 최신 도구, 실습 워크북, 통합 워크플로우  

---

## 🎯 주요 변경사항

### 1. 3부작 구조로 재편성
**v12.0**: 단일 문서 (6개 섹션, ~1,150 lines)  
**v13.0**: 3부작 구조 (총 ~6,700 lines)
- Part 1: 기초 편 (Context Engineering + Markdown + AI 기본) - 2,224 lines
- Part 2: 고급 도구 편 (Copilot Workbook + MCP + SpecKit) - 1,845 lines
- Part 3: 통합 워크플로우 편 (실제 연구 프로젝트) - 2,635+ lines

**이유**: 점진적 학습, 선택적 학습 경로, 깊이 있는 내용

### 2. 2025년 최신 도구 반영
**v12.0**: 기본 AI 도구만 (Copilot, ChatGPT)  
**v13.0**: 2025년 연구 도구 생태계 통합
- **GitHub Copilot 2025 기능**: Agent mode, Custom Instructions, Multi-model, Vision, Next Edit Suggestions
- **MCP (Model Context Protocol)**: task-master-mcp, arxiv-mcp, zotero-mcp, notion-mcp, jupyter-mcp
- **SpecKit**: 7단계 워크플로우 (선택적)
- **2025 문헌 조사 도구**: Elicit, Perplexity Research, Consensus, Scite, ResearchRabbit, Connected Papers, Semantic Scholar
- **노트 도구**: NotebookLM (Mind Map, Audio Overview)

**이유**: 2025년 연구 워크플로우 트렌드 반영, 도구 선택권 제공

### 3. 실습 워크북 추가
**v12.0**: 개념 설명 중심, Markdown 예제 위주  
**v13.0**: 4개 실습 Exercise (Part 2 Section 2)
- **Exercise 1**: 문헌 조사 자동화 (키워드 생성, 논문 요약 템플릿)
- **Exercise 2**: 연구 계획서 작성 (구조 생성, 섹션별 초안)
- **Exercise 3**: 데이터 분석 스크립트 (Python/R 코드, 비코더 친화적)
- **Exercise 4**: 논문 작성 지원 (Methods, Results 섹션)

**이유**: 실습 중심 학습, 즉시 적용 가능한 기술

### 4. 전공별 완전 시나리오
**v12.0**: 교육학 예시 1개 (간단)  
**v13.0**: 5개 전공 완전 시나리오 (Part 3 Sections 2-6)
- **인문사회**: 교육학 석사논문 (20-25 pages)
- **자연과학**: 생명과학 실험 연구 (18-22 pages)
- **공학**: 컴퓨터공학 시스템 개발 (18-22 pages)
- **사회과학**: 사회학 질적 연구 (18-22 pages)
- **예체능**: 음악학 연주 논문 (15-18 pages)

**이유**: 전공별 맞춤 가이드, 실제 프로젝트 구조 제공

### 5. 리소스 파일 대폭 확장
**v12.0**: 리소스 없음 (본문에 통합)  
**v13.0**: 43+ 리소스 파일
- **Part 1**: 11개 (학생 가이드, 설정, 템플릿, 참조, 예시)
- **공유**: 21개 (계획, MCP, SpecKit, 템플릿, 프롬프트, 비교표)
- **Part 2**: 3개 (워크북, MCP 설치, SpecKit 설치)
- **Part 3**: 14개 (시나리오, 도구 생태계, 루틴, 추적, 품질, 성공 사례)

**이유**: 모듈화, 재사용성, 빠른 참조

### 6. 데모 파일 체계화
**v12.0**: 데모 파일 없음  
**v13.0**: 8개 데모 폴더 + 지원 파일
- **기존 4개**: VS Code, Copilot, Projects, task-master-mcp
- **신규 4개**: Copilot 워크북, MCP 설치, SpecKit, 통합 워크플로우
- **지원**: rehearsal-guide, demo-cheatsheet, backup-plan, master-demo-script, DEMO-QUICK-START

**이유**: 강의 시연 품질 향상, 리허설 지원, 백업 계획
