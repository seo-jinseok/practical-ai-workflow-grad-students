## 🚀 빠른 시작

### 전체 과정 학습 (8-10시간)
1. **24번**으로 통합 워크플로우 전체 그림 이해 (30분)
2. **25번**으로 2025 도구 생태계 파악 (1-2시간)
3. **26-29번** 중 자신의 전공 시나리오 집중 학습 (2-3시간)
4. **30번**으로 문헌 조사 도구 7개 통합 활용법 습득 (1-2시간)
5. **31-34번**으로 실전 루틴 및 품질 관리 방법 적용 (1-2시간)
6. **35번**으로 실제 폴더 구조 및 파일 예시 확인 (30분)

### 상황별 활용법

**📖 논문 연구 처음 시작하는 경우**
- 24 → 26(자신의 전공) → 31 → 35 순서로 학습
- 실제 폴더 구조를 복사해서 프로젝트 시작

**🔍 기존 연구에 AI 도구 적용하고 싶은 경우**
- 25 → 30 → 32 → 33 순서로 학습
- 도구별 특성과 통합 방법에 집중

**⏰ 시간 Management 개선하고 싶은 경우**
- 31 → 32 → 34 순서로 학습
- 주간 루틴과 품질 관리에 집중

**🤔 연구 중 문제 상황에 빠진 경우**
- 33 → 37 순서로 참조
- 문제 해결 프로토콜과 트러블슈팅 가이드 활용

**💡 다른 사람의 성공 사례를 보고 싶은 경우**
- 36번만 학습
- 동기부여와 실전 팁에 집중
