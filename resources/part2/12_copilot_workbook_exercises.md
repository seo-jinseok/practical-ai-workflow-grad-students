## 🎯 워크북 개요

### 📋 전체 목표
GitHub Copilot을 활용하여 **연구 워크플로우를 완전히 자동화**하는 방법을 배우는 실용적 워크북입니다. 4개의 Exercise를 통해 문헌 조사부터 논문 작성까지 전 과정을 AI와 함께 수행하는 법을 습득합니다.

### ⏰ 예상 소요 시간: 2-3시간
- 각 Exercise: 30-45분
- 실습 시간 + 휴식 시간 포함

### 🎓 필요 사전 지식
- Part 1 완료 (Markdown, AI 기본 사용법)
- VS Code 기본 사용법
- GitHub Copilot 계정 (학생 무료)

### 📁 준비물
```
📁 사전 준비물
├── 💻 VS Code 설치 및 Copilot 활성화
├── 🗂️ 새 폴더 생성: `copilot-workbook/`
└── 🎯 연구 주제 선정 (Exercise용)
```

## 📊 4개 Exercise 전체 지도

### 🗂️ Exercise 구성

| Exercise | 제목 | 예상 시간 | 주요 결과물 | 난이도 |
|----------|------|-----------|-------------|--------|
| **1** | 문헌 조사 자동화 | 30-45분 | 검색 키워드 + 논문 요약 템플릿 | ⭐⭐☆☆☆ |
| **2** | 연구 계획서 작성 | 30-45분 | 연구 계획서 초안 (목차 + 3개 섹션) | ⭐⭐☆☆☆ |
| **3** | 데이터 분석 스크립트 | 30-45분 | Python 분석 코드 + 시각화 | ⭐⭐⭐☆☆ |
| **4** | 논문 작성 지원 | 30-45분 | Methods + Results 섹션 초안 | ⭐⭐⭐☆☆ |

### 🔗 Exercise 간 연결성

```
Exercise 1 → Exercise 2 → Exercise 3 → Exercise 4
   ↓           ↓           ↓           ↓
 키워드       계획서       분석        논문
```

**단계별 완성 결과물**:
- **Ex1 완료**: 체계적 문헌 조사 체계
- **Ex1+2 완료**: 완전한 연구 계획서
- **Ex1+2+3 완료**: 연구 계획 + 데이터 분석 체계
- **전체 완료**: 논문 작성까지 완료

## 🏃‍♀️ Exercise 1: 문헌 조사 자동화

### 🎯 목표
- **Copilot으로 연구 주제에 맞는 검색 키워드 생성**
- **논문 요약 템플릿 작성**
- **실제 논문 1편을 템플릿으로 요약**

### 📋 준비 단계 (5분)

#### ✅ 준비 체크리스트
- [ ] VS Code 실행 및 Copilot 활성화 확인
- [ ] 새 폴더 생성: `copilot-workbook/`
- [ ] 새 파일 생성: `copilot-workbook/literature-search.md`
- [ ] 연구 주제 선정 (예시: "온라인 학습 참여도", "ESG 투자", "창의적 사고" 등)

#### 💡 연구 주제 예시 (전공별)
- **교육학**: 온라인 학습, 학습 동기, 게이미피케이션, 원격 교육
- **공학**: 나노 소재, 인공 지능, IoT 보안, 신재생 에너지
- **인문학**: 한국 현대문학, 문화 정체성, 디지털 HUMANITIES, 창작론
- **사회과학**: 복합재난, 고용 구조 변화, 사회복지 정책, 지역社会发展
- **자연과학**: 기후변화, 생물다양성, 힐 코어 (CRISPR), 천체물리학

#### 🔍 Step 2: 검색 키워드 생성 (10분)

**🎯 지침**:
1. `## 검색 키워드` 제목을 입력하세요.

2. 다음 줄에 `- ` 입력 후 Tab 키를 누르세요.

3. Copilot이 키워드를 제안하면:
   - **적합**: Tab 키로 수락
   - **부적합**: Esc 키로 거부 후 다른 제안 대기

4. 10-15개 키워드가 나올 때까지 반복하세요.

5. 영문 키워드도 추가하세요:
   ```markdown
   ### English Keywords
   - [키워드 입력 후 Tab]
   ```

**📝 완성된 예시 결과**:
```markdown
## 검색 키워드

- 온라인 학습
- 학습자 참여도
- 원격 교육
- 이러닝
- 학습 동기
- 자기조절학습

- 디지털 교육
- 가상교실

- 교육 기술
- 온라인 수강

### English Keywords
- online learning

- distance education
- e-learning
- learning motivation

- learning flow
- interaction
- learning satisfaction

- online course
```

#### 📄 Step 3: 논문 요약 템플릿 작성 (10분)

```markdown

### 논문 1: [제목]
- **저자**: [저자명] ([연도])
- **저널**: [저널명], [권호]
- **연구 목적**: [핵심 연구 질문]
- **방법론**: [연구 설계 및 방법]
- **주요 발견**: [핵심 결과 2-3개]
- **시사점**: [연구의 의의 및 한계]
- **내 연구와 연결**: [이 논문이 내 연구에 주는 시사점]
```

2. Copilot이 템플릿 구조를 제안하는지 관찰하세요.

3. 템플릿을 완성한 후, 실제 논문 1편을 이 템플릿으로 요약해보세요.

**💡 논문 찾는 방법**:
- Google Scholar에서 한글/영문 키워드로 검색
- 近5년 내published 논문 우선
- 인용수가 높은 논문 선택

```python
# 3. 그룹 비교 분석
# 성별에 따른 참여도 비교
male_engagement = data[data['gender'] == 'M']['engagement_score']
female_engagement = data[data['gender'] == 'F']['engagement_score']

t_stat, p_value = stats.ttest_ind(male_engagement, female_engagement)
print(f"성별 참여도 비교: t={t_stat:.3f}, p={p_value:.3f}")

# 시각화
data.boxplot(column='engagement_score', by='gender')
plt.title('성별에 따른 참여도 비교')
plt.suptitle('')  # 기본 제목 제거
plt.show()
```
