# 2025 AI 프롬프트 컬렉션

## 목적
- 대학원 연구에 최적화된 AI 프롬프트 라이브러리
- 연구 생명주기 각 단계별 맞춤형 AI 도구 활용 가이드
- AI 도구별 최적화된 프롬프트 템플릿 및 사례 모음

## 연구 생명주기별 AI 프롬프트

### 1. 연구 기획 단계 (Research Planning)

#### 연구 주제 탐색 프롬프트
```
Copilot 프롬프트:
"연구 주제: [구체적 연구 분야]
목표: 해당 분야의 최신 트렌드와 미개척 영역 식별
요구사항:
- 최근 3년간 주요 연구 동향 분석
- 연구 격차(gaps) 및 기회 영역 식별
-可行的 연구 방향 3-5개 제시
- 각 방향의 예상 난이도 및 기대 성과 평가

AI 도구 연동:
- Google Scholar API 활용
- 키워드 기반 트렌드 분석
- 인용 네트워크 분석"
```

#### 연구 설계 프롬프트
```
MCP 서버 프롬프트:
"연구 질문: [구체적 연구 질문]
목표: 연구 방법론 및 설계 제안
요구사항:
- 적합한 연구 접근법 (정량/정성/혼합)
- 샘플 크기 및 샘플링 방법 추천
- 데이터 수집 도구 및 기법 제시
- 윤리적 고려사항 및 IRB 준비사항
- 예상 연구 기간 및 마일스톤 계획

Task Master 연동:
- 연구 계획서를 작업 관리에 등록
- 마일스톤 및 의존성 관계 설정
- 진도 추적 체계 구축"
```

### 2. 문헌 리뷰 단계 (Literature Review)

#### 체계적 문헌 검색 프롬프트
```
Copilot 프롬프트:
"연구 주제: [구체적 주제]
목표: 체계적 문헌 검색 전략 수립
요구사항:
- 효과적인 키워드 조합 (한국어/영어) 제안
- 주요 학술 데이터베이스 목록
- Boolean 검색식 작성
- 문헌 포함/제외 기준 설정
- 예상 검색 결과 수 및 품질 평가

MCP 서버 연동:
- 자동화된 문헌 검색 수행
- 중복 문헌 식별 및 제거
- 메타데이터 추출 및 정리"
```

#### 문헌 분석 및 요약 프롬프트
```
Task Master 프롬프트:
"문헌 목록: [파일 경로 또는 목록]
목표: 문헌의 체계적 분석 및 요약
요구사항:
- 각 문헌의 핵심 내용 5-7줄 요약
- 연구 방법론 및 주요 발견 정리
- 한계점 및 향후研究方向 식별
- 문헌 간의 관계 및 상호보완성 분석
- 메타분석 또는 연구통합 가능성 평가

AI 분석 도구:
- 텍스트 마이닝 및 토픽 모델링
- 인용 네트워크 분석
- 공기어 분석 및 트렌드 파악"
```

### 3. 데이터 수집 단계 (Data Collection)

#### 설문지 개발 프롬프트
```
Copilot 프롬프트:
"연구 목적: [구체적 연구 목적]
목표: 효과적인 설문지 개발
요구사항:
- 연구 질문에 기반한 개념 작동 정의
- 측정 항목 및 문항 구성
- Likert 척도 또는 다양한 응답 형식 추천
- 설문지의 타당성 및 신뢰도 확보 방안
- 파일럿 테스트 계획

AI 도구 활용:
- 기존 검증된 척도 추천
- 설문 문항의 명확성 및 편향성 검토
- 응답자 부담 최소화 방안"
```

#### 인터뷰 가이드 개발 프롬프트
```
MCP 서버 프롬프트:
"연구 주제: [구체적 주제]
목표: 체계적인 인터뷰 가이드 작성
요구사항:
- 반구조화 인터뷰 질문 구성
- 주요 프로빙 질문 및 후속 질문 목록
- 인터뷰 진행 순서 및 시간 배분
- 윤리적 고려사항 (동의서,保密 등)
- 인터뷰 기록 및 전사 계획

AI 도구 연동:
- 인터뷰 음성 파일 자동 전사
- 주제별 코딩 및 테마 분석
- 핵심 인사이트 자동 추출"
```

### 4. 데이터 분석 단계 (Data Analysis)

#### 정량적 데이터 분석 프롬프트
```
Copilot 프롬프트:
"데이터: [데이터 설명]
분석 목표: [구체적 분석 목표]
요구사항:
- 데이터 전처리 및 정제 방법
- 적합한 통계 분석 방법 추천
- 가설 검정 절차 및 해석
- 시각화 도구 및 그래프 유형
- 결과의 통계적 유의성 평가

Python 코드 예시:
- pandas, numpy, scipy, matplotlib 활용
- 분석 결과의 자동 보고 생성
- 재현 가능한 분석 파이프라인 구축"
```

#### 정성적 데이터 분석 프롬프트
```
Task Master 프롬프트:
"데이터 유형: [인터뷰, 관찰, 문서 등]
분석 접근: [테마 분석, 내용 분석, 담화분석 등]
목표: 체계적인 정성적 데이터 분석
요구사항:
- 코딩 체계 및 코드북 개발
- 테마 식별 및 패턴 발견 방법
- 신뢰도 및 타당성 확보 방안
- 연구자 편향 최소화 전략
- 결과의 타당화 및 검증

AI 도구 활용:
- 텍스트 마이닝 및 NLP
- 주제 모델링 (LDA, BERT 등)
- 자동화된 코딩 및 테마 추출"
```

### 5. 논문 작성 단계 (Paper Writing)

#### 섹션별 논문 작성 프롬프트
```
Copilot 프롬프트:
"논문 섹션: [서론/방법론/결과/토론/결론]
연구 배경: [연구 주제 및 문맥]
목표: 해당 섹션의 효과적 작성
요구사항:
- 섹션별 구조 및 내용 가이드라인
- 핵심 주장 및 증거 제시 방법
- 학술적 문체 및 표기법
- 참고문헌 인용 및 관리
- 논리적 흐름 및 연결성 확보

AI 도구 연동:
- Grammarly를 통한 문체 교정
- 참고문헌 관리 (Zotero, Mendeley 연동)
- 표 및 그림 자동 생성"
```

#### 논문 검토 및 개선 프롬프트
```
MCP 서버 프롬프트:
"초고 파일: [파일 경로]
목표: 논문의 종합적 검토 및 개선
요구사항:
- 논리적 구조 및 흐름 검토
- 명확성, 정확성, 완성도 평가
- 방법론의 적절성 및 재현성 검토
- 결과 해석의 객관성 평가
- 학술적 윤리 및 표절 검토

AI 도구 활용:
- Turnitin을 통한 표절 검사
- 자연어 처리를 통한 일관성 검토
- 메타데이터 및 인용 네트워크 분석"
```

### 6. 학술 발표 단계 (Academic Presentation)

#### 프레젠테이션 설계 프롬프트
```
Copilot 프롬프트:
"논문 내용: [핵심 연구 내용]
청중: [학술대회/학회/일반인 등]
목표: 효과적인 학술 발표 자료 제작
요구사항:
- 청중 특성에 맞는 내용 구성
- 핵심 메시지 3-5개 선정
- 시각적 자료 (차트, 그래프) 활용
- 발표 시간 및 Q&A 전략
- 기술적 Demonstrations 포함

AI 도구 활용:
- 슬라이드 자동 생성 (Canva, PowerPoint)
- 발표 스크립트 및 타이밍 가이드
- 인터랙티브 요소 및 멀티미디어 통합"
```

## 프롬프트 엔지니어링 고급 기술

### 컨텍스트 엔지니어링
- **상세한 배경 정보**: 연구 분야, 방법론, 목표를 명확히 제시
- **도메인 지식 포함**: 전문 용어 및 개념 적절히 활용
- **제약 조건 명시**: 시간, 예산, 리소스 제약사항 포함

### 출력 형식 지정
- **구조화 출력**: JSON, 표, 리스트 등 명확한 형식 지정
- **예시 제공**: 원하는 출력 형태의 예시 포함
- **평가 기준**: 품질 평가 기준 및 지표 명시

### 반복적 개선
- **피드백 루프**: AI 응답에 대한 피드백 제공
- **단계적 구체화**: 초기 답변을 바탕으로 더 구체적인 질문
- **다양한 관점**: 다른 각도에서 동일한 질문 시도
