# 주간 보고서 템플릿 2025

## 목적
- 연구 진행 상황의 체계적 기록 및 추적을 위한 AI 기반 템플릿
- 연구자의 주간 성과와 계획을 한 눈에 파악할 수 있는 보고서
- Task Master와 연동된 자동화된 진도 관리 시스템

## AI 프롬프트: 주간 성과 분석
```
주차: [ week n ]
연구 주제: [구체적 연구 주제]
목표: AI를 활용한 주간 성과 및 계획 수립
AI 도구 활용:
- Copilot: 성과 데이터 분석 및 인사이트 추출
- Task Master: 작업 완료율 및 지연 작업 식별
- 출력 형식: 핵심 성과, 계획, 이슈, 개선사항
```

## 1. 주요 성과 섹션 (5-7줄)
### 완료된 작업
- **문헌 리뷰**: 관련 논문 15편 검토 및 핵심 내용 요약 완료
- **데이터 수집**: 설문 응답 250명 (목표 500명의 50%) 수집 완료
- **방법론 설정**: 연구 설계의 통계적 검정력 분석 완료

### 정량적 지표
- **진도율**: 전체 프로젝트의 35% 완료 (목표: 40%)
- **논문 작성**: 초안 2개 섹션 완성 (서론, 문헌고찰)
- **품질 점수**: Task Master 평가 8.5/10 (이전 주 대비 0.3 향상)

### AI 도구 활용 성과
```
Copilot 프롬프트:
"이번 주 완료된 연구 작업들의 성과와 품질을 분석하고,
각 작업별 AI 도구 활용 효율성을 평가해주세요.
특히 [구체적 작업]에서 AI 도구가 기여한 부분을 
구체적으로 분석해주세요."
```

## 2. 다음 주 계획 섹션 (5-7줄)
### 우선순위 작업
- **데이터 분석**: 수집된 설문 데이터의 기초 통계 분석 (Priority 1)
- **인터뷰 진행**: 연구 대상자 5명 심층 인터뷰 수행 (Priority 2)
- **논문 작성**: 방법론 섹션 초안 작성 (Priority 3)

### 예상 소요 시간
- **데이터 분석**: 15시간 (AI 도구 활용으로 기존 대비 40% 단축)
- **인터뷰**: 10시간 (설문 설계 포함)
- **논문 작성**: 8시간 (Copilot 작성 지원)

### AI 도구 활용 계획
```
Task Master 프롬프트:
"다음 주의 주요 작업을 Task Master에 등록하고,
각 작업의 예상 소요 시간과 의존성 관계를 분석해주세요.
AI 도구 활용으로 단축 가능한 시간과 
우선순위 조정 필요성도 포함해주세요."
```

## 3. 직면한 이슈 섹션 (5-7줄)
### 기술적 문제
- **데이터 품질**: 설문 응답 중 15%가 불완전하거나 부적절한 응답
- **AI 도구 오류**: Task Master 서버의 주기적인 연결 중단 문제
- **해결 방안**: 데이터 정제 AI 도구 도입 및 MCP 서버 설정 수정

### 연구적 도전
- **표본 대표성**: 특정 지역/계층의 응답이 부족함
- **방법론 검증**: 적용한 통계 방법의 타당성에 대한 의구심
- **해결 방안**: 추가 샘플링 및 방법론 전문가 컨설팅

### AI 도구 문제해결
```
MCP 서버 프롬프트:
"연구 과정에서 발생한 문제와 해결 과정을 분석하고,
AI 도구 활용으로 예방할 수 있는 방법을 제시해주세요.
특히 [구체적 문제]에 대한 AI 기반 예방 시스템 구축 방안도 포함해주세요."
```

## 4. 학습 인사이트 섹션 (5-7줄)
### 새로운 발견
- **연구 방법론**: 혼합 방법론의 효과성이 기존 예상보다 높음
- **AI 도구 활용**: Copilot의 코드 자동완성 기능이 연구 속도를 30% 향상
- **협업 패턴**: 원격 연구팀의 AI 도구 활용 시 Synchronous 워크플로우가 효과적

### 개선 포인트
- **데이터 수집**: AI 기반 설문 최적화로 응답률 25% 향상 가능
- **문헌 정리**: 자동 분류 도구를 활용한 효율성 개선 필요
- **진도 관리**: 기존 방식 대비 Task Master의 마일스톤 관리 기능 활용 증가

### Task Master 프롬프트
```
Task Master 프롬프트:
"이번 주의 연구 과정에서의 학습 인사이트를 정리하고,
AI 도구 활용으로 얻은 경험과 지식을 
다음 주 작업 계획에 어떻게 반영할지 
구체적으로 제안해주세요."
```

## 5. 주간 리뷰 및 피드백 섹션 (5-7줄)
### 성과 평가
- **목표 달성도**: 설정된 주간 목표의 85% 달성 (매우 양호)
- **품질 지표**: 연구 결과의 신뢰성과 재현성 평가 8.2/10
- **AI 도구 효율성**: 전체 연구 시간의 30% 절감 효과 확인

### 개선 제안
- **프로세스 최적화**: 데이터 분석 자동화 도입으로 시간 절약 20% 예상
- **품질 향상**: Peer review 시스템 구축으로 연구 결과의 객관성 강화
- **팀워크 강화**: AI 도구를 활용한 실시간 협업 환경 개선

### 다음 주 전략
- **집중 영역**: 데이터 분석 및 결과 해석에 최고 우선순위 부여
- **AI 도구 활용**: SpecKit을 활용한 연구 문서화 프로세스 도입
- **리스크 관리**: 예상되는 기술적 문제에 대한 선제적 대응 방안 마련
