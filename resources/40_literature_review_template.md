# 문헌 리뷰 템플릿 2025

## 목적
- 체계적이고 포괄적인 문헌 리뷰 수행을 위한 AI 기반 템플릿
- 연구 영역의 최신 동향과 핵심 이론을 체계적으로 정리
- AI 도구를 활용한 효율적인 문헌 검색, 분석, 요약 프로세스

## AI 프롬프트: 문헌 검색 및 분류
```
연구 주제: [구체적 연구 주제]
목표: 관련 문헌 100편 수집 및 5개 카테고리 분류
AI 도구 활용:
- Copilot: 키워드 기반 검색 전략 수립
- Task Master: 문헌 분류 및 진행 상황 추적
- MCP 서버: 자동화된 문헌 수집 및 메타데이터 추출
```

## 1. 키워드 전략 섹션 (5-7줄)
### 주요 키워드 설정
- **핵심 키워드**: [주요 개념 1], [주요 개념 2], [주요 개념 3]
- **영문 키워드**: "keyword 1", "keyword 2", "keyword 3"
- **전문 용어**: [전문 용어 1], [전문 용어 2], [전문 용어 3]
- **연관 개념**: [연관 개념 1], [연관 개념 2]

### 검색 전략
- **데이터베이스**: PubMed, Web of Science, Scopus, Google Scholar
- **시간 범위**: 2019-2024 (최근 5년간)
- **언어**: 영어, 한국어
- **문서 유형**:期刊 논문, 학술서적, 연구보고서

### AI 도구 활용
```
Copilot 프롬프트:
"연구 주제 [구체적 주제]에 대한 문헌 검색을 위해
효과적인 키워드 조합과 Boolean 검색식을 제안해주세요.
특히 [구체적 연구 분야]의 주요 저널과 컨퍼런스도 포함해주세요."
```

## 2. 문헌 수집 섹션 (5-7줄)
### 수집 기준
- **편수**: 목표 100편 (고품질 논문 50편, 중급 논문 50편)
- **피인용 횟수**: 평균 피인용 10회 이상 (저자는 5회 이상)
- **학술적 엄격성**: Peer-reviewed 학술지 논문 우선
- **관련성**: 연구 주제와의 직접적 관련성 80% 이상

### 자동화 도구
- **메타데이터 추출**: AI 기반 논문 제목, 저자, 초록 자동 추출
- **중복 제거**: 연구 주제와 방법론 기준으로 중복 논문 식별 및 제거
- **품질 평가**: AI 모델을 통한 논문 품질 점수 자동 평가

### AI 도구 활용
```
MCP 서버 프롬프트:
"설정된 검색 기준으로 문헌을 수집하고,
각 논문의 메타데이터와 품질 점수를 분석해주세요.
중복 문헌을 식별하고, 연구 주제와의 관련성도 
평가해주세요. 분류는 5개 카테고리로 나누어주세요."
```

## 3. 요약 형식 섹션 (5-7줄)
### 기본 정보
- **제목**: [논문 제목]
- **저자**: [제1저자 et al.]
- **학술지**: [학술지명], [연도], [권호]
- **피인용**: [피인용 횟수회]
- **연구 유형**: [정량/정성/혼합 방법론]

### 핵심 내용 요약
- **연구 질문**: [핵심 연구 질문]
- **방법론**: [사용된 연구 방법]
- **주요 발견**: [핵심 결과 3-5개]
- **한계점**: [연구의 한계]
- **향후 연구**: [제시된 향후 연구 방향]

### AI 도구 활용
```
Task Master 프롬프트:
"각 문헌의 핵심 내용을 5-7줄로 요약하고,
연구 질문, 방법론, 주요 발견, 한계점, 향후研究方向을 
체계적으로 정리해주세요.
분류 기준: 이론적 기여, 방법론적 혁신, 실증적 발견, 
문헌 gaps, 후속 연구 가능성"
```

## 4. AI 방법론 섹션 (5-7줄)
### 자동화 분석 기법
- **텍스트 마이닝**: 키워드 빈도 분석 및 토픽 모델링
- **인용 네트워크**: 저자 인용 관계 및 연구 그룹 분석
- **추세 분석**: 연구 주제의 연도별 발전 추세 파악
- **공기어 분석**: 동시에 나타나는 키워드 패턴 분석

### AI 도구 활용 단계
1. ** preliminares 처리**: PDF 텍스트 추출 및 전처리
2. **주제 모델링**: LDA (Latent Dirichlet Allocation) 적용
3. **감정 분석**: 연구자의 의견 및 태도 분석
4. **패턴 인식**: 연구 방법론 및 결과의 패턴 추출

### AI 도구 활용
```
Copilot 프롬프트:
"수집된 문헌에 대한 AI 기반 분석을 수행하고,
연구 주제별 트렌드, 주요 연구 그룹, 핵심 방법론, 
미개척 영역을 식별해주세요.
특히 AI가 수행할 수 있는 자동화 분석 기법과 
그 결과에 대한 인간 검증 과정을 포함해주세요."
```

## 5. 문헌 분류 및 체계화 섹션 (5-7줄)
### 카테고리 분류
- **이론적 기반** (20편): 핵심 개념과 이론 정리
- **방법론적 접근** (25편): 연구 방법과 기법 비교
- **실증적 연구** (30편): 데이터 기반 연구 결과
- **기술적 혁신** (15편): 새로운 도구나 기술 활용
- **정책 및 실무** (10편): 실무 적용 사례 및 정책 제언

### 분류 기준
- **핵심성**: 연구 주제와의 직접적 관련성
- **신규성**: 기존 연구와의 차별성
- **신뢰성**: 연구 방법론의 타당성
- **영향력**: 해당 분야에 미친 영향

### AI 도구 활용
```
Task Master 프롬프트:
"문헌을 5개 카테고리로 분류하고, 각 카테고리의 
특성과 중요도를 분석해주세요.
문헌 간의 논리적 관계와 상호 보완성을 파악하고,
연구 격차(gaps)와 future research directions을 
식별해주세요. 결과는 카테고리별 요약 표로 정리해주세요."
```

## 6. 한계점 및 향후研究方向 섹션 (5-7줄)
### 기존 연구의 한계
- **방법론적 한계**: 실험 설계, 샘플링, 데이터 수집의 한계
- **이론적 한계**: 개념 정의, 이론적 프레임의 불완전성
- **실증적 한계**: 일반화 가능성, 인과관계 규명의 한계
- **실용적 한계**: 실제 적용 가능성, 정책 제언의 구체성

### AI 도구가 채울 수 있는 공백
- **데이터 처리**: 대용량 데이터의 자동화 처리 및 분석
- **패턴 인식**: 복잡한 데이터 패턴의 AI 기반 식별
- **예측 모델링**: 미래 트렌드 및 결과 예측
- **통합 분석**: 다양한 관점의 데이터 통합 분석

### 향후 연구 방향
- **방법론 발전**: AI 도구를 활용한 새로운 연구 방법론 개발
- **현장 적용**: AI 도구의 실제 연구 활용 사례 확장
- **윤리적 검토**: AI 활용의 연구 윤리 및 투명성 확보
- **교육 및 확산**: 연구자 대상 AI 도구 활용 교육
