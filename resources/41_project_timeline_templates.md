# 프로젝트 타임라인 템플릿

## 목적
- 연구 프로젝트의 단계별 시간 계획과 마일스톤 관리
- AI 도구 통합을 통한 효율적인 시간 관리
- 프로젝트 진행 상황 추적 및 조정

## 표준 연구 프로젝트 타임라인 (6개월)

### 1-2주차: 프로젝트 설정
- 연구 계획서 작성
- AI 도구 환경 구축
- 문헌 수집 시작

### 3-8주차: 문헌 리뷰 및 방법론
- 체계적 문헌 검토
- 연구 설계 구체화
- IRB 승인 절차

### 9-16주차: 데이터 수집
- 설문/인터뷰 진행
- 실험 수행
- 데이터 품질 관리

### 17-20주차: 분석 및 작성
- 통계 분석 수행
- 논문 초안 작성
- 결과 검토 및 수정

## 마일스톤 관리
- **M1**: 문헌 리뷰 완료 (8주차)
- **M2**: 데이터 수집 완료 (16주차)
- **M3**: 분석 완료 (19주차)
- **M4**: 논문 제출 (24주차)

## AI 도구 활용 타이밍
- **초기**: Copilot, VS Code 설정
- **중기**: Task Master, MCP 연동
- **후기**: AI 분석, 논문 작성 지원
