# AI 어시스턴트 프롬프트 라이브러리

## 목적
- 연구工作中 활용할 수 있는 AI 프롬프트 모음
- Copilot, Task Master, MCP 서버와의 협업 최적화
- 연구 품질 향상을 위한 프롬프트 엔지니어링

## 연구 단계별 프롬프트

### 문헌 리뷰 프롬프트
```
주제: [연구 주제]
목표: 관련 문헌 50편 조사 및 주요 발견 5개 추출
출력 형식: 
- 연구 제목 및 저자
- 주요 방법론
- 핵심 결과
- 본 연구와의 관련성 (1-2문장)
- 한계점

주제별 분류:
1. 이론적 배경
2. 방법론적 접근
3. 실증적 결과
4. 한계점 및 향후 연구
```

### 데이터 분석 프롬프트
```
데이터: [데이터 설명]
목표: [분석 목표]
요청:
1. 기본 통계 요약
2. 시각화 (히스토그램, 상자도, 산점도)
3. 가설 검증 결과
4. 비즈니스/학술적 해석

AI 도구 활용:
- 코드는 Python으로
- 설명은 한글 및 영문
- 시각화는 matplotlib/seaborn
```

### 논문 작성 프롬프트
```
섹션: [논문 섹션명]
문맥: [연구 배경, 선행연구]
목표: [해당 섹션의 목표]
 字数: [목표 字数]

포함 사항:
- 핵심 내용 3-4개 포인트
- 선행연구와의 연결고리
- 본 연구의 독창성
- 한계점과 향후 연구 방향

형식: 학술 논문 표준 형식
참고문헌: APA 7th edition
```

## AI 도구별 최적화 프롬프트

### Copilot 전용 프롬프트
```markdown
"코드 작성"
"문서 작성" 
"문서 리뷰"
"이해도 검증"
```

### Task Master 전용 프롬프트
```markdown
"작업 분해"
"마일스톤 설정"
"진행 상황 추적"
"우선순위 조정"
```

### MCP 서버 전용 프롬프트
```markdown
"서버 상태 확인"
"데이터베이스 검색"
"외부 API 연동"
"결과 저장"
```

## 프롬프트 개선 기술

### 명확성 향상
- 구체적인 요청사항 명시
- 출력 형식 지정
- 예시 제공

### 컨텍스트 제공
- 프로젝트 배경 설명
- 이전 대화 내용 참조
- 관련 문서 링크

### 품질 검증
- 결과 확인 요청
- 대안 제시 요청
- 개선 제안 요청
