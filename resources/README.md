## 🎯 전체 목적과 비전

이 중앙 허브는 **Part 1(기초) → Part 2(고급) → Part 3(통합)**의 학습 여정을 한 곳에서 내비게이션하고, 2025년 최신 AI 도구(Copilot, MCP, SpecKit, Elicit, Perplexity 등)를 연구 워크플로우에 통합하는 방법을 제공합니다.

- **문제 해결**: 기존 Poor Discoverability → 완전한 파일 트리와 퀵스타트로 해결
- **접근성**: 초보자/중급자/고급자, 전공별 맞춤 경로 제공
- **통합성**: 모든 Parts와 공유 리소스를 하나의 체계로 연결
- **실용성**: 바로 사용할 수 있는 템플릿과 예시 포함
