# GitHub Copilot 학생 무료 혜택 신청 가이드

**목적**: GitHub Student Developer Pack을 통해 Copilot Pro를 무료로 사용하는 방법

**소요 시간**: 약 10-15분

**필요 사항**: 학교 이메일 주소, 학생증 또는 재학증명서

## 1. Free vs Pro 비교

| 기능 | Free Tier | Pro (학생 무료) |
|------|-----------|----------------|
| 코드 완성 횟수 | 월 2,000회[^1] | **무제한** |
| Premium 요청 | 50회/월[^2] | **300회/월**[^3] |
| AI 모델 선택 | 제한적 | **전체 모델**[^4] |
| Chat 기능 | 제한적 | 전체 |
| Agent mode | ❌ | ✅ |
| Custom Instructions | ❌ | ✅ |
| 월 비용 | 무료 | **무료 (학생)** / $10 (일반)[^3] |

**학생 혜택 요약**:
- ✅ Copilot Pro 전체 기능 무료
- ✅ 모든 AI 모델 사용 (GPT-5, Claude Sonnet 4.5, Gemini 2.5 Pro, Grok)
- ✅ 무제한 코드 완성
- ✅ Agent mode로 복잡한 작업 자동화
- ✅ 졸업 후에도 재학 중에는 계속 무료

## 2. 신청 절차 (Step-by-step)

### Step 1: GitHub 계정 생성 (이미 있다면 Skip)
1. https://github.com 방문
2. "Sign up" 클릭
3. 이메일 주소 입력 (개인 이메일 권장, 학교 이메일은 다음 단계에서)
4. 계정 생성 완료

### Step 2: GitHub Education Pack 신청
1. https://education.github.com/pack 방문
2. "Get your pack" 또는 "Sign up for Student Developer Pack" 클릭
3. GitHub 로그인 (Step 1에서 만든 계정)

[SCREENSHOT: GitHub Education Pack 메인 페이지]

### Step 3: 학생 인증
1. **학교 이메일 주소** 입력
   - 보통 `@university.ac.kr`, `@edu` 등으로 끝나는 이메일
   - 학교 이메일이 없다면: 학생증 + 재학증명서 업로드로 인증 가능

2. **학교 정보** 입력
   - 학교 이름 (영문)
   - 졸업 예정 연도
   - 재학 중인 과정 (석사/박사)

[SCREENSHOT: 학생 인증 폼]

3. **사용 목적** 작성
   - "I plan to use GitHub Education for..."
   - 예시: "I will use GitHub Copilot for my graduate research in education technology and data analysis."
   - 간단히 2-3문장으로 작성

4. **학생 증명 서류 업로드** (학교 이메일 인증 실패 시)
   - 학생증 사진 (학생 번호, 이름, 학교 이름 보이게)
   - 또는 재학증명서 스캔본
   - PDF, JPG, PNG 형식

### Step 4: 제출 및 대기
1. "Submit your application" 클릭
2. 이메일 확인 (학교 이메일로 인증 링크 전송)
3. 인증 링크 클릭하여 확인 완료
4. 승인 대기: **보통 1-3일 소요**
   - 빠르면 몇 시간 내 승인
   - 학교 이메일 도메인이 등록되어 있으면 즉시 승인
   - 서류 검토 필요 시 며칠 소요 가능

## 3. Copilot Pro 활성화

### Step 1: 승인 확인
1. 승인 이메일 확인 (GitHub에서 발송)
2. https://education.github.com/discount_requests 에서 상태 확인
3. "Approved" 표시 확인

### Step 2: Copilot Pro 활성화
1. VS Code에서 Copilot 확장 설치 (아직 안 했다면)
   - Extensions (Ctrl+Shift+X / Cmd+Shift+X)
   - "GitHub Copilot" 검색
   - Install 클릭

2. GitHub 계정으로 로그인
   - VS Code 하단 상태바에서 Copilot 아이콘 클릭
   - "Sign in to GitHub" 선택
   - 브라우저에서 인증 완료

3. Pro 혜택 확인
   - 상태바 Copilot 아이콘에 체크 표시
   - 마우스 오버 시 "GitHub Copilot Pro" 표시 확인

[SCREENSHOT: Copilot Pro 활성화 상태]

### Step 3: 기능 테스트
1. 새 파일 생성 (.md 또는 .py)
2. 타이핑 시작하면 제안 표시되는지 확인
3. Chat 열기 (Ctrl+Shift+I / Cmd+Shift+I)
4. 모델 선택 드롭다운에서 여러 모델 확인

## 4. 혜택 유지 방법

### 매년 재인증
- GitHub Education Pack은 **1년 단위**[^5]로 갱신
- 만료 1개월 전 이메일 알림
- 재학 중이면 같은 절차로 재신청
- 졸업하면: Pro → Free tier로 자동 전환 (또는 유료 구독)

### 재인증 절차
1. 만료 알림 이메일 받음
2. https://education.github.com/pack 재방문
3. "Renew your pack" 클릭
4. 재학 증명 (졸업 예정일 업데이트 또는 새 서류 제출)
5. 승인 후 자동 연장

### 인증 실패 시
- 학교 이메일 주소 변경되었는지 확인
- 재학증명서 최신 것으로 재제출
- GitHub Education Support에 문의: education@github.com

## 5. 문제 해결 (Troubleshooting)

### ❌ 학교 이메일이 인식되지 않을 때
**원인**: 학교 도메인이 GitHub Education에 미등록

**해결**:
1. 학교 이메일 대신 **학생증 + 재학증명서** 업로드로 인증
2. 또는 학교 도메인 등록 요청: https://github.com/education/schools
3. 대체 인증 방법: ISIC 카드 (국제학생증) 사용 가능

### ❌ 학생 인증이 거부될 때
**원인**: 서류 불명확 또는 정보 불일치

**해결**:
1. 학생증 사진 다시 찍기 (고화질, 정보 선명하게)
2. 재학증명서는 **발급일 3개월 이내** 것 사용
3. 영문 증명서 권장 (한글은 번역 필요 시 시간 소요)
4. 재신청 전 며칠 대기 후 다시 시도

### ❌ Pro 혜택이 적용되지 않을 때
**원인**: VS Code에서 계정 연동 미완료

**해결**:
1. VS Code에서 로그아웃 후 재로그인
   - Command Palette (Ctrl+Shift+P)
   - "GitHub Copilot: Sign Out"
   - 다시 로그인
2. VS Code 재시작
3. Copilot 확장 재설치
4. GitHub 웹사이트에서 Copilot 설정 확인: https://github.com/settings/copilot

### ❌ "You've reached your limit" 메시지
**원인**: Free tier로 인식되어 월 2,000회 한도 도달

**해결**:
1. Education Pack 승인 상태 재확인
2. https://github.com/settings/copilot 에서 plan 확인
3. "Pro" 표시 없으면: Education Support에 문의
4. 임시 해결: 다음 달까지 대기 또는 다른 AI 확장 사용

### ❌ 졸업 후 혜택 종료
**상황**: 졸업하면 학생 혜택 자동 만료

**옵션**:
1. **Free tier로 전환** (월 2,000회)
2. **Pro 유료 구독** ($10/월)
3. **대안 확장** 사용 (Cline, Roo Code 등 자체 서버 무료 모델)

## 6. 추가 혜택 (Student Developer Pack)

Copilot Pro 외에도 다음 도구들이 무료로 제공됩니다:

| 도구 | 혜택 | 용도 |
|------|------|------|
| **GitHub Pro** | 무제한 private repo | 코드 저장 |
| **Azure Credits** | $100/년 | 클라우드 서버 |
| **Heroku** | 무료 호스팅 | 웹 앱 배포 |
| **Namecheap** | 무료 도메인 (.me) | 개인 사이트 |
| **MongoDB** | Atlas 무료 크레딧 | 데이터베이스 |
| **Stripe** | 수수료 면제 | 결제 시스템 |
| **Canva Pro** | 1년 무료 | 디자인 도구 |
| **JetBrains** | 전체 IDE 무료 | 개발 도구 |

전체 목록: https://education.github.com/pack/offers
